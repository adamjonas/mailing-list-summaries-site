Summary:

Adam, a member of bitcoin-dev mailing list, shared his thoughts on the use-case of having an offline seed manager that can be backed up once and support multiple wallets. He also noted that using bip39 mnemonic seed as an interface between an offline seed manager and a hardware or software wallet is convenient. Rodolfo Novak from Coinkite Inc. announced plans to implement the Deterministic Entropy From BIP32 Keychains BIP on Coldcard and asked if there was any planned BIP number and review of the BIP yet. Ethan mentioned his plan to submit a BIP for deterministic entropy from BIP32 keychains. In an email conversation with Rodolfo, he considered splitting the BIP into two parts: one for the entropy derivation and another for the applications section. A user has opened a BIP PR at https://github.com/bitcoin/bips/pull/910 and created a Python library with test vectors referenced in the BIP text. Christopher Allen suggested a review by cryptographic engineering expert Pieter Wuille, and suggested using the offline tool #LetheKit provided by BlockchainCommons/bc-lethe-kit as a solution.
