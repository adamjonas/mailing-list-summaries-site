Summary:

The development of the Signet proposal (BIP-325) has seen progress due to the efforts of ajtowns. The recent update allows block signatures to be verified as transactions, making it compatible with PSBT and eliminating the need for the "simple signature verifier/creator" feature. However, this also means that Signet will undergo a hard fork. Despite this, it is not a concern as the platform is not yet being used seriously. The BIP pull request is available on Github for feedback.
