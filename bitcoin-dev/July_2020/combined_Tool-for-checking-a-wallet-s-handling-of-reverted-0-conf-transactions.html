Summary:

There is an issue with several wallets that cannot handle reverted transactions at zero confirmations, resulting in an increased balance being shown to users without decreasing it when a transaction is replaced. This has also caused users to be unable to spend other valid coins. KZen-networks has developed a new tool called big-spender, available on Github, which allows users to test the behavior of their own wallet and learn more about the issue. It is important for users to test their wallets using this tool to ensure that they are not affected by the issue.
