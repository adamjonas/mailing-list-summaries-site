Summary:

The Signet proposal (BIP-325) has made progress thanks to ajtowns. Block signatures can now be verified as transactions, which means it is probably compatible with PSBT and no longer requires the custom "simple signature verifier/creator" stuff. However, this also means that Signet will undergo a hard fork. It is not considered to be an issue since no one is using it seriously yet. The BIP pull request can be found on Github for feedback.
