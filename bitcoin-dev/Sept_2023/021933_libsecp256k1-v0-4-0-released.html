Summary:

We are pleased to announce the release of version 0.4.0 of libsecp256k1. The main feature in this update is the addition of the ellswift module, which introduces the ElligatorSwift encoding. This new feature enhances the functionality of the library and provides users with more options for encoding and decoding data.To learn more about the changes and improvements in version 0.4.0 and earlier releases, you can refer to the full release notes available on GitHub. The release notes provide a comprehensive overview of all the updates and bug fixes implemented in each version. You can access the release notes at the following link: https://github.com/bitcoin-core/secp256k1/blob/master/CHANGELOG.mdWe encourage all users of libsecp256k1 to upgrade to the latest version to take advantage of the new features and improvements.
