Summary:

The email conversation is about a patch that was made to adapt one line of code to support boost filesystem version 3. The patch has been tested and it was confirmed that it works on older versions too, including Boost 1.41.0. Jeff Garzik, who works for exMULTI, Inc., responded to the change made by an unknown individual. He mentioned that the change appears to be technically correct but expressed disappointment in the use of #ifdefs that have made the code messier. Jeff acknowledged that it's only one line and can be removed when old APIs are removed.Luke-Jr added that it's only one line and can also be removed when the old APIs are sufficiently removed. However, Jeff suggested that a nicer version would be better. Most projects work to hide this stuff in headers so that the actual code is cleaner.In conclusion, the patch was provided as an attachment in the email conversation. Jeff Garzik's email address is jgarzik@exmulti.com.
