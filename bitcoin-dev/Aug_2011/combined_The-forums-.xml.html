Summary:

Bitcoin developers have been discussing the negative impact of the forum.bitcoin.org due to illegal activities and inappropriate comments made there. Some members suggest removing the link altogether from bitcoin.org and using an alternative medium to address user issues. Jeff Garzik has proposed replacing the current forum with another more suitable platform for soliciting user feedback on UI features. Chris Acheson has suggested using Stack Exchange as a new forum to replace the current one. There are concerns about the negative influence of the forum on bitcoin, with some users engaging in illegal activities and giving the project a bad name.The developers hope to launch a new site by the end of August but need more people to commit to using it. They also mention concerns regarding bitcoin.org DNS and who has access to it. Some members support removing the forum entirely due to the illegal activity taking place on it, such as selling a bitcoin mining virus. The main complaint is that there are too many inappropriate topics, such as illegal activity, politics, and philosophy, which are not helpful to users.Some propose creating an official help forum dedicated to client software and protocol and how to use/integrate/modify it, with a link to the FAQ before posting. However, given the kind of people bitcoin attracts, maintaining a normal forum may be unrealistic. There have been suggestions for a new forum that is focused on helping users adopt the technology, without political or philosophical discussions. One developer has pushed for the Bitcoin Stack-Exchange as the main user/merchant support site, which has defined acceptable/unacceptable topics in advance and has a good user moderation/reputation system.Vladimir has created a new community portal at https://bitcoin.org.uk which contains advanced forum functionality, blogging platform, community calendar and chat. It is professionally hosted on high-end dedicated servers running FreeBSD in a DC located in Amsterdam, NL. Additionally, if any Bitcoin businesses would like to have their “support” forums set up there so that they retain moderatorial control over it please let him know. Soft launch is scheduled on 15th of August, but it is fairly functional already and open for registration. The discussion also touches upon renaming the project, but concludes by distinguishing an implementation from the network itself. The developers hope to create a simpler forum structure that is easier to moderate and focuses on helping users. However, they acknowledge that moderation power is limited due to the lack of people who can mod such a forum. There have been proposals to create new categories such as Technical Support and Development, but others argue that these are also difficult to moderate.
