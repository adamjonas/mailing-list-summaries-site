Summary:

Venkatesh Srinivas has suggested removing an obsolete flag from the UNIX makefile in a patch message. The flag in question is -DFOURWAYSSE2 and it is no longer required. In the patch message, changes made to the makefile.unix file are included. The DEFS variable is updated to remove the aforementioned flag and replace it with -DUSE_SSL. Additionally, the USE_UPNP variable is set to 0 and the LIBS variable is defined. Jeff Garzik, who works at exMULTI, Inc., applied the suggestion made by Venkatesh Srinivas. There is no further information or details about the changes made in the patch message.
