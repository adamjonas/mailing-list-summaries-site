Summary:

The attached patch which has been sent by Venkatesh Srinivas, defines MSG_NOSIGNAL to 0 on platforms where it is unavailable. Previously this definition was only available under __WXMSW__. The patch is provided in the form of a diff file and contains changes for util.h file. T* alignup(T* p) function has been defined in the same file. In the updated version, #ifndef MSG_NOSIGNAL has been added along with #define MSG_NOSIGNAL 0. Additionally, #ifdef __WXMSW__ has been used to define MSG_DONTWAIT as 0 and UINT64_MAX as _UI64_MAX. The file has undergone 4 insertions and 1 deletion in total.
