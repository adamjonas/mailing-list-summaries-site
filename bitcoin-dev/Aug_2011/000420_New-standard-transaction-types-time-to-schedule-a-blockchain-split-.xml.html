Summary:

In an email exchange on August 25th, 2011, Gregory Maxwell and Pieter Wuille discussed the size of bitcoin output scripts. Wuille provided estimates for various types of scripts, including a single address, 1-out-of-2, 2-out-of-3, and (a and b) or c. These estimates ranged from 21 bytes to 67 bytes for the address itself and from 27 bytes to 88 bytes for the script. Maxwell remarked that the 187-byte size for a destination with six keys and complicated rules was "pretty darn good."
