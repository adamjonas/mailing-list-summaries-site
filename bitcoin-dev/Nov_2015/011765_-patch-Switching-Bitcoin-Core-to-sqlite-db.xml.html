Summary:

On November 15, 2015, Peter R via bitcoin-dev wrote about exploring the idea of "meta-cognition" with Bitcoin Unlimited. The group discussed the possibility of an automatic fork to the longest chain if Bitcoin Unlimited gets stuck and cannot prove if a block is valid or invalid. This situation cannot be ignored as it affects the processing of block results. For example, the BDB/LevelDB issue caused BDB to fail accepting further changes to the UTXO set which meant that there was no way to validate the next block or any new transactions.
