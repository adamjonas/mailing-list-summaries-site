Summary:

Bitcoin Core, the software which is used for cryptocurrency transactions, has appointed Jonas Schnelli as the new GUI maintainer. Schnelli has been involved in redesigning icons, network statistics, and improving user experience for the past year. He has also been unofficially giving direction on GUI matters, making his appointment official. As the new GUI maintainer, he will handle GUI-related issues on the GitHub tracker, and assist in merging GUI-related pull requests. This development is seen as a significant step towards addressing the much-needed improvement in the user experience of Bitcoin Core. Schnelli, a senior software developer from Switzerland, has been active in this area for the last year, redesigning all the icons for 0.11.0, visualizing various network statistics, and continuously improving user experience. His appointment is expected to improve the usability of Bitcoin Core.
