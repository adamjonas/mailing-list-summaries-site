Summary:

Bitcoin Core version 0.10.4rc1 binaries are now available for download from the website, which includes the source code. The release is a minor one with bug fixes and changes such as the BIP65 (CLTV) consensus change and relay policy preparation for BIP113. A preliminary note about the release can be found on GitHub and report bugs using the issue tracker at Github. The release candidates are test versions of releases, and if no critical problems are found, it will be tagged as 0.10.4.
