Summary:

Bitcoin Core version 0.11.2rc1 binaries are now available for download from bitcoin.org/bin/bitcoin-core-0.11.2/test/. The source code can be found on Github under the signed tag at github.com/bitcoin/bitcoin/tree/v0.11.2rc1. This is a new minor version release that includes bug fixes, the BIP65 (CLTV) consensus change, and relay policy preparation for BIP113. Preliminary release notes for the 0.11.2 release can be found at github.com/bitcoin/bitcoin/blob/0.11/doc/release-notes.md.Release candidates are test versions of releases, and when no critical problems are found, this release candidate will be tagged as 0.11.2. Bugs should be reported using the issue tracker at github.com/bitcoin/bitcoin/issues. The message also includes a PGP signature.
