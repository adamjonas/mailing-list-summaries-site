Summary:

Bitcoin Core version 0.10.4rc1 binaries, including the source code, can now be downloaded from the website. The release is a minor one with bug fixes and changes such as the BIP65 (CLTV) consensus change and relay policy preparation for BIP113. A preliminary note about the release can be found on GitHub. Users can report any bugs using the issue tracker at Github. It is important to note that release candidates are test versions of releases. If no critical problems are found, this release will be tagged as 0.10.4.
