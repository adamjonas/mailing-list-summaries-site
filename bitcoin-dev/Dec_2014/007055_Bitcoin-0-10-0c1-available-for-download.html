The release candidate 1 of version 0.10.0 of Bitcoin has been made available for download at bitcoin.org. The binaries are ready to be used by anyone interested in testing the new release. The preliminary release notes, which provide detailed information about the changes made and new features added in this version, can be found on Github. Users are encouraged to report any bugs that they encounter while using the new release by using the issue tracker on Github.
