Summary:

In the world of cryptocurrency, one can provide a reusable payment code (BIP-47) instead of an actual address. However, this is not yet supported by the clients/apps most people use. This would be less of a hurdle than providing a service that generates addresses from xpub. According to a message on bitcoin-dev, it is not possible to specifically not support address reuse on a blockchain where anyone can send funds without permission required. Therefore, multiple payments to the same address should always be expected.
