Summary:

Shigeya Suzuki reported strange results when running src/test/test_bitcoin on Debian 9. Test cases were conducted on three platforms including OS X High Sierra 10.13.6, Debian 9 and Ubuntu 16.04. The first two platforms were tested with ./configure --enable-debug and the third one was tested with ./configure --with-debug --disable-wallet. Configuration 2 caused a core dump while others did not. The error message displayed during configuration 2 test cases revealed that it may have been due to toolchain differences.Marco Falke via bitcoin-dev acknowledged that this was a known issue in test code that is only compiled when debugging. He assured that it will be fixed in the upcoming 0.16.2 and 0.17.0 releases. The technical issues related to the Bitcoin Core code base are tracked on their issue tracker. Shigeya Suzuki had doubts if the issue was due to toolchain differences.
