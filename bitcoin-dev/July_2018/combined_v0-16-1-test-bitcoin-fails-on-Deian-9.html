Summary:

Shigeya Suzuki encountered an issue while running the src/test/test_bitcoin test on Debian 9. This problem was not observed when running the test on OS X High Sierra 10.13.6 or Ubuntu 16.04. The issue occurred specifically when using Configuration 2, resulting in a core dump during the execution of 264 test cases.Marco Falke from bitcoin-dev acknowledged that this issue was already known and is related to the test code, which is compiled only when debugging. He assured that the problem would be addressed in the upcoming releases, specifically in versions 0.16.2 and 0.17.0.To keep track of technical issues related to the Bitcoin Core code base, there is an issue tracker available.
