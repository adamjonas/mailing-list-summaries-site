Summary:

In 2013, Bitcoin developer Gavin Andresen wrote three Bitcoin Improvement Proposals (BIPs) based on a preliminary payment protocol specification. BIP 0070 defines the network protocol and messages, BIP 0071 defines MIME types for the messages, and BIP 0072 defines the bitcoin URI extension. A question was raised about the possibility of having a URI that is only a payment request, with an example given as "bitcoin:?request=https%3A%2F%2Fmerchant.com%2Fpay.php%3Fh%3D2a8628fc2fbe".
