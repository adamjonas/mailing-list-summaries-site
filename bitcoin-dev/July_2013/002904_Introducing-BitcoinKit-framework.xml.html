Summary:

The launch of BitcoinKit.framework has been announced, a Cocoa framework that enables developers to create Bitcoin wallet applications for Mac OS X. The framework uses bitcoind and provides a small API for developer use. It will soon support other Bitcoin implementations, such as libcbitcoin. Hive, a new Mac wallet app, will be the first application to use BitcoinKit as its backbone. The source code can be found on GitHub, and support is available through GitHub issues and a Bitcointalk thread. A sample GUI app is also included.
