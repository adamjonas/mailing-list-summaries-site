Summary:

BitMail is a secure peer-to-peer email service that enables users to send emails from friend to friend without relying on a central server. The platform is built on the Echo Protocol and features key/repleo exchange, chat and instant messaging capabilities, and strong end-to-end multi-encryption using PGP-kind/AES over SSL with libgcrypt. BitMail also integrates with Libspoton and offers an additional security layer with the GB-feature for emails. The platform prevents data retention and is independent of a web of trust. It uses HTTP and HTTPS connections and is open-source under the BSD license.
