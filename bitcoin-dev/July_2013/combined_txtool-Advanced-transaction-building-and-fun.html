Summary:

Txtool is a command line tool built in node.js that allows for building complex bitcoin transactions. It was created by Jeff Garzik, Senior Software Engineer and open source evangelist at BitPay, Inc., to demonstrate advanced bitcoin features and make it easier for users to experiment with them. The intended audience has a basic understanding of bitcoin transactions. Txtool interfaces with Bitcoin-Qt/bitcoind. On GitHub, the full list of commands and theory of operation can be found. Some of the working examples available on txtool include multi-signature transaction building and passing around transactions for signing.Txtool aims to demonstrate advanced bitcoin features and make it easier for users to experiment with them.
