Summary:

There is a growing discussion around secure peer-to-peer email systems that do not rely on a central server. One such system is Key/Repleo-Exchange, which uses the Echo Protocol to create a fully decentralized email network with built-in chat and instant messaging features. The software is open source under the BSD license and includes Libspoton integration and an additional security layer with the GB-Feature for emails.However, there are concerns about similar software called "GoldBug" that has been posted about on various forums by apparent sock accounts. Pond is another email-like messaging system that runs over Tor and provides forward security resistant to traffic analysis. It also utilizes a small file for users to grant someone permission to message them. While Pond takes forward security seriously, it is written in Go.BitMail is another secure peer-to-peer email service that enables users to send emails from friend to friend without relying on a central server. The platform is built on the Echo Protocol and features key/repleo exchange, chat and instant messaging capabilities, and strong end-to-end multi-encryption using PGP-kind/AES over SSL with libgcrypt. BitMail also integrates with Libspoton and offers an additional security layer with the GB-feature for emails.The Trusted Platform Module (TPM) is a hardware-based security solution used to store and manage cryptographic keys and sensitive data. Pond uses the TPM chip to reliably destroy keys for data that an SSD device might have otherwise made un-erasable. It is worth noting that BitMail has a server running for testing purposes at BitMail.sf.net. The IP address of the server is 178.83.35.133:4710. It is unclear how long the server will be running, but it should be available for the next few days.
