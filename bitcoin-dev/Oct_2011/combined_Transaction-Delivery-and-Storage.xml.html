Summary:

Brian McQueen inquired about the creation, storage, and transmission of bitcoin transactions and how these transactions would be reintroduced into the peer-to-peer (P2P) network once they were outside it. The response was that the P2P network is not essential to bitcoin and that with patches, transactions can be imported and exported to text from a regular client. The distributed algorithm and block chain are more important to bitcoin than the P2P network. It was even suggested that all bitcoin systems could be interconnected with Morse code operators running spark gap transmitters without any IP packets, and it would still be bitcoin.The Contracts wiki page discusses transactions passed around outside the P2P network and how transactions are not broadcast right away. Specialized software that interacts with the Bitcoin network directly will implement many things on the contracts page. Transactions will be moved around by having clients do HTTP POSTs of protocol buffers to servers that are listening and know how to interpret the received messages.
