Summary:

The Bitcoin-Qt version 0.5 Release Candidate 1 is about to be released with an improved GUI. However, all the translations for the old wxWidgets Bitcoin are obsolete and new translations need to be written. Three translations already exist in the ".ts" files stored in the src/qt/locale folder. The QT Linguist tool or an online tool like Transifex could be used to create new translations. Someone needs to coordinate translation efforts, determine if Transifex is a good tool to use and write documentation to make it easy to create and submit new translations.
