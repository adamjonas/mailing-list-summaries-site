The email chain discussed various topics related to Bitcoin development and translation efforts. One message explained that generated coins must wait 120 blocks before they can be spent, and if another node generates a block within a few seconds of yours, it may occasionally cause the block to become "not accepted" and not spendable. Another message pointed out duplicated strings in the Bitcoin software and questioned their purpose. The discussion also touched on the use of custom software for creating work in pools and the need for stateless TCP/IP "stacks" to deal with DDoS attacks. Finally, there was a mention of Transifex, a tool used for coordinating translation efforts for Bitcoin.In terms of translations, the Bitcoin-Qt version 0.5 Release Candidate 1 with an improved GUI is set to be released soon, and new translations are needed as all the translations for the old wxWidgets Bitcoin are obsolete. Three translations already exist: de nl and ru. The translations are stored in ".ts" files in the src/qt/locale folder, and the 'QT Linguist' tool can be used to create translations. An online tool like Transifex could also be used to crowd-source the work, and somebody must coordinate translation efforts, figure out if Transifex is a good tool to use, and write some documentation to make it easy for people to create and submit new translations. Some strings are duplicated, which could be improved upon. Additionally, someone is needed to manage translation efforts, evaluate the use of Transifex tool and create documentation to make it easier for people to create and submit new translations.Geir Harald Hansen pointed out the issue of duplicated strings in Bitcoin translations, and John Smith agreed to look into the matter further. Chris also noted that the same string may have to be translated in different ways depending on the context they appear in. Some users expressed interest in contributing Norwegian Bokmål and Italian translations.The email chain also discussed adding Norwegian Bokmål as a language for translation.
