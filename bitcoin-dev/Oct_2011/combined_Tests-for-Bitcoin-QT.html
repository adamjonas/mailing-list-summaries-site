Summary:

The Bitcoin QT is a framework that worked excellently the first time on a Mac. It was easy to pull, compile and run as compared to using the makefile and CLI. However, there are no unit tests for the Qt GUI code itself as of this moment. The tests for the bitcoin core are still present in src/tests, but they need to be built with the makefile. Qmake is currently only used for the GUI executable.
