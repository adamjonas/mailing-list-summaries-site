Chris, a software developer from Tawhakisoft.com, is wondering if it would be a good idea to have an alias to wallet ID nameserver in the client software. The purpose of this would be to allow individuals to use an English name to describe a wallet public key address, which the software can then use to look up the wallet ID. Chris feels that wallet IDs are hard to remember and recall, hence the need for an alternative solution.
