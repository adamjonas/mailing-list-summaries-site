Summary:

In 2013, Matt Corallo announced the release of version 0.8.5 of Bitcoin which included a new category and was available for download from the Bitcoin Github repository. Kip Warner, a software engineer, thanked Matt for the update and provided his contact information, including a preference for OpenPGP encrypted/signed mail. Recently, in a mailing list post, Kip Warner proposed refining the "Office" category in bitcoin-qt application's XDG desktop integration to "Finance," citing that XDG's menu specification allows for it. He suggested modifying the line in bitcoin-qt.desktop to read "Categories=Office;Finance;" but mentioned his weak knowledge of Git and did not provide a patch himself. In response to this proposal, Matt announced the pull request at https://github.com/bitcoin/bitcoin/pull/2999 which includes the new category. The discussion took place on the Bitcoin-development mailing list. The bitcoin-qt application's XDG desktop integration on some desktop environments requests that it be placed under the "Office" menu category, which is a rather broad category. The suggestion to refine this further into "Finance" seems to be welcomed by the community, given that XDG's menu specification allows for this. The author apologizes for not providing the patch himself due to his weak knowledge of Git.
