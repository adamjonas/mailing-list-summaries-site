Summary:

Aymeric Vitte has released node-Tor, a JavaScript implementation of the Tor protocol for servers and browsers, as open-source software under an MIT license. The project aims to enable the building of Tor-based projects from browsers and servers. It does not add nodes to the Tor network or implement Tor Browser features but provides Onion Proxy and Onion Router functionality directly in the browser. Node-Tor can establish circuits with other nodes that understand the Tor protocol, act as a Tor node, and work with Tor network nodes. The project was funded by NLnet via EU Horizon 2020 Next Generation Internet Privacy & Trust Enhancing Technologies and is small, fast, and extensively tested for video streaming applications. It is pure JS and not a browser extension/module. Future plans for node-Tor include implementing elliptic crypto, integrating connections via WebRTC Snowflake, and linking it with "Discover and move your coins by yourself" for anonymous blockchain search and transaction sending from the browser.
