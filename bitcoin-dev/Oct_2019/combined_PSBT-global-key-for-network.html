Summary:

Jimmy Song proposed adding a network key to BIP174, Partially-Signed Bitcoin Transactions. He suggests adding a key-value pair to indicate the network that the coins are on, with the key proposed being 0x03 for network in the global key-value store. The value would be a variable integer with 0x00 indicating Bitcoin mainnet and 0x01 indicating Bitcoin testnet. Other coins that want to use the PSBT should use the coin network number from SLIP-0044 with the high bit removed. Each signer should ensure that the inputs being referenced in the PSBT exist except for a Proof-of-Reserves input, making it crucial to know which network the coins are on.Jon replied to the email suggesting that adding a network section would not help verify anything and giving a false sense of security is actually a security minus. Currently, BitcoinJS only uses network parameters to allow for the use of addresses in addOutput, but Jon is considering removing this feature. Nevertheless, the proposal could potentially be extended to other networks.
