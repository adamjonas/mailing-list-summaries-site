Summary:

The author discusses the trade-offs between confidentiality and base layer performance in architecture, particularly in light of recent global monetary affairs. They express a shift towards greater confidentiality over performance due to regulatory changes. The author also questions whether Ring sigs with Stealth addresses impede blockchain performance and if they align with Bitcoin's longer-term development roadmap. In an email forwarded to the Bitcoin Dev mailing list, Aymeric Vitte shares the news that node-Tor is now open source under an MIT license. Node-Tor is a JavaScript implementation of the Tor protocol on server-side and inside browsers, which could potentially be used to anonymize Bitcoin apps such as light wallets. The project is financed by NLnet via EU Horizon 2020 Next Generation Internet Privacy & Trust Enhancing Technologies and is modular, fast, and has been extensively tested for video streaming. Possible future steps include implementing elliptic crypto and connections via WebRTC Snowflake.
