A pull request to the bips repository for Signet has stalled due to uncertainty surrounding whether or not Signet should have a BIP at all. The maintainer is questioning whether or not Signet is Bitcoin. However, the argument is that Signet is indeed Bitcoin and therefore deserves a BIP. Having a BIP would facilitate interoperability between different software in the Bitcoin space. Feedback is welcomed on the pull request itself or through other channels.
