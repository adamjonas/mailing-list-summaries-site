Summary:

Payward Inc., a financial technology company, has submitted a draft to the Internet Engineering Task Force (IETF) proposing the creation of an Internet MIC (IMIC). The proposed IMIC aims to identify internet-based financial markets in a way that is compatible with the existing Market Identification Code (MIC) standard set by the International Organization for Standardization (ISO). The draft can be found on the IETF's website.
