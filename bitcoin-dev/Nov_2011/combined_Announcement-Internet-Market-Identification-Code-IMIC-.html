Summary:

Payward Inc. has submitted a draft to the IETF for an Internet MIC (IMIC) which identifies internet-based financial markets. The IMIC is designed to be superset-compatible with the ISO's existing Market Identification Code standard. Walter Stanish, of Payward Inc., submitted the abstract for the draft. The draft can be found on the datatracker.ietf.org website.
