Summary:

Gavin Andresen, the Bitcoin developer, reported that there was an issue holding up a 0.5.0/0.4.1 final release: some bug reports of crashes on startup when re-encrypting previously encrypted wallets were received on Linux64 systems. He tried reproducing it on Ubuntu 10.10 server and wasn't able to reproduce it. Gavin asked for help in figuring out what is going on and whether only bitcoin-qt that is crashing or Ubuntu is OK but other Linux variations are broken. Thiago replied with his experience of trying to reproduce the error by doing specific actions on Ubuntu 11.10 64bits. He found that automatic wallet upgrade was working properly and didn't face any crashes.
