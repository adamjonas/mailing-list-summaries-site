Summary:

Bitcoin developers are discussing the naming of the client protocol identifier. The identifier is not a public name and is only used by developers. One suggestion is to name it Satoshi, as homage to the person who made the original client reference protocol. However, there is confusion about what "Satoshi 0.5" refers to since the wx GUI client has been removed and replaced with Bitcoin-Qt.
