Summary:

Over the weekend, Bitcoin's rc2 was updated on https://bitcoin.org/bin/0.9.3/test/. The new version includes additional fixes to improve the network and memory behavior of Bitcoin. One change that was made is the removal of a useless millisleep in the socket handler. This will help to improve the overall efficiency of the network. Additionally, stricter memory limits have been placed on CNode to prevent potential issues with memory usage and ensure system smoothness without any unnecessary hiccups.The update also features improvements to orphan transaction handling. `-maxorphantx=` and `-maxorphanblocks=` options have been added to give users greater control over the maximum orphan counts. Overall, these updates aim to enhance the functionality and performance of Bitcoin's rc2, helping it run more efficiently and effectively.
