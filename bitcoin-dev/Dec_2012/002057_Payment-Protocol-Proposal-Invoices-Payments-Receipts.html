In a discussion thread, Mike Hearn pointed out the absurd design of SMTP that makes it impossible to start a paragraph with the word "From." However, Roy chimed in to clarify that it's not an issue with SMTP but rather a problem with the file format of the standard BSD UNIX mailbox. The mailbox uses lines beginning with "From " to delimit messages, causing confusion when encountering the same sequence within a message.
