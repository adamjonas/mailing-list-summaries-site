Summary:

A recent pull request has been submitted on GitHub for the Bitcoin repository, which proposes updates to the COIN_DUST constant. The proposal suggests a decrease in the COIN_DUST value from 0.01 BTC to 0.001 BTC due to the increase in bitcoin value, currently standing at US$13.67. This change aims to reduce the threshold at which dust spam is considered. In addition to this, the proposal also includes changes to the TX miner and relay fee defaults. The suggested defaults are 0.001 BTC and 0.0005 BTC respectively. The proposed update reflects the growth of dust spam in the unspent transaction output dataset. Miners using this code will need to include transactions with outputs equal to or greater than 0.0005 BTC and require a fee greater than or equal to 0.001 BTC to relay transactions with outputs of 0.0005 BTC. Overall, the review of impact shows a reduction in the definition of COIN_DUST and changes to the required fees for miners to include and relay transactions.
