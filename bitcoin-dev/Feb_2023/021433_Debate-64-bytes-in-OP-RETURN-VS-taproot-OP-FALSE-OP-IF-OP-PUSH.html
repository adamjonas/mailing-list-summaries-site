Summary:

On February 5, 2023, Aymeric Vitte via bitcoin-dev raised a concern regarding the number of OpReturn outputs. He suggested that allowing only one OpReturn output causes problems in composing different uses of OpReturn and thus, any number of OpReturn outputs should be allowed. Previously, when he created the bitcoin-transactions nodejs module, the limit from the specs was 512B.Moreover, there is no reason to put a size limit on OpReturn outputs considering the various other ways people can publish data, including with a 75% discount. Therefore, it is advisable to let the fee market deal with it.
