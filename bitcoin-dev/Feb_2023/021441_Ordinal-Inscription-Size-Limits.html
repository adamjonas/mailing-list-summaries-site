In the past, images were stored in Bitcoin using large numbers of multisig output addresses that encode the images. This made it trivial to store the images in a way that they looked like legitimate transactions. However, introducing this type of censorship into Bitcoin would go against the project's goals and is considered fundamentally undesirable.
