Summary:

To make a transaction with two outputs, the first output is for payee while the second one is for the fee to be given to the person who transmits the transaction to the network. The transaction should be signed and printed along with the private key controlling the address of output B. Afterward, it needs to be sent in the mail to one or more greedy people who are unknown to you.Regarding Namecoin, it allows mixing outputs for name registration with pay to hash outputs. This means that Namecoin can be used for anonymous publishing through posts.
