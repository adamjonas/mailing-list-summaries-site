In response to feedback, Krzysztof Okupski has announced that the spec for Bitcoin will now be under version control. The spec can be found on GitHub at https://github.com/minium/Bitcoin-Spec. It demonstrates a commitment to transparency and collaboration within the Bitcoin community.
