Summary:

On July 14, 2014, Krzysztof Okupski announced that the Bitcoin specification will now be under version control due to invaluable feedback. The spec can be found on GitHub at https://github.com/minium/Bitcoin-Spec. The old link to the PDF will also be kept updated as well. This move is seen as a way to increase the number of Bitcoin developers in the long run by making the whole system more understandable to more people.
