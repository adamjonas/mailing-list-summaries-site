Summary:

If you're building on OSX and have trouble with the new dependency, libtool, after updating to git HEAD, Gavin Andresen suggests installing it via homebrew (brew install libtool) or port (port install libtool). The full list of dependencies can be found in doc/build-osx.md. After installing libtool, run ./autogen.sh and configure with your desired options. In his case, Gavin uses "./configure --disable-hardening --disable-silent-rules CXXFLAGS='-g3 -O0 -DDEBUG_LOCKORDER'".
