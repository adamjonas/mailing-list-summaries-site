Summary:

Pieter Wuille, a Bitcoin Core developer, has created a new segwit testnet with activation via BIP9, as well as support for BIP68, BIP112, and BIP113. This development is especially notable because it supports the Lightning Network, a payment protocol that could help Bitcoin scale. Those interested are encouraged to try it out by spinning up a node. The source code can be found on Pieter's Github repository, while feedback can be given on the #segwit-dev channel on Freenode.
