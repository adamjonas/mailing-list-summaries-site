Summary:

Pieter Wuille, a Bitcoin Core developer, has developed a new segwit testnet with activation via BIP9 and support for BIP68, BIP112, and BIP113. The new development supports the Lightning Network, which is a payment protocol that could help Bitcoin scale. Those who are interested can try it out by spinning up a node. The source code of this development can be found on Pieter's Github repository. Feedback related to this development can be given on the #segwit-dev channel on Freenode.
