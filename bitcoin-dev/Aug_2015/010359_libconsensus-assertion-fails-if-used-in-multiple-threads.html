Tamas Blummer has confirmed that libconsensus is working correctly within the Bits of Proof stack, in line with test cases borrowed from Bitcoin Core to verify the Java implementation of the script engine. He found that the performance of libconsensus is surprisingly close to the java version, and that validating a 2-of-2 a multi-sig transaction runs at 1021 ops/sec with java and 1135 ops/sec in libconsensus.
