Summary:

The email thread revolves around the issue of centralization in Bitcoin mining and the requirement for testing data for BIP101. Pieter Wuille's original email highlights the centralized nature of mining as a problem, to which Gavin Andresen seeks clarification on why it's a problem. Venzen Khaosan explains that this is a significant issue for Bitcoin, and development has not stopped at any point in the past. He argues that miners are still users who have guaranteed coinbase, but only when they start proving themselves useful in a free-floating fee market absent coinbase subsidy can the topic be revisited.Later in the conversation, Gavin Andresen mentions that as Bitcoin grows, pieces of the ecosystem will specialize, and Satoshi's original code did everything, but that is changing, and that is okay. Venzen Khaosan requests XT 8MB hard-fork testing data mentioned for BIP100. The email thread ends with Benjamin Franklin's quote about learning from being ignorant.
