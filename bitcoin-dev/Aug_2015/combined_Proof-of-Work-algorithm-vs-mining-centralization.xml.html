Summary:

The proposal suggests a solution to protect Bitcoin mining from centralization by periodically changing the proof of work algorithm. This change would discourage the specialization of hardware and enable competition using general-purpose computing machines. The list of hash algorithms would be indexed, and most elements initially would be the current proof of work algorithm so that the existing mining community could solve most blocks. Only one algorithm every so often would be available to everyone else. Eventually, the ratio would fall, giving miners time to convert their expertise into more productive activities.The proposal aims to prevent a situation where a group of people create hardware optimized for mining that makes it too expensive for others to compete, leading to centralization. The variable proof of work algorithm would enable general-purpose computing machines to be the best device for mining Bitcoin, providing fair competition.
