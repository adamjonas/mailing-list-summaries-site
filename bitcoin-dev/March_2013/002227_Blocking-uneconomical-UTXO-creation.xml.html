The discussion is about the use of UTXO in a peer-to-peer network and how it can be shifted to a Distributed Hash Table (DHT) for sharing the load of storing, validating, and verifying. While the current use of UTXOs in the Satoshi client does not directly point towards DHT, it does speed up validation by checking if an output is unspent. An alternative way to bootstrap and validate transactions exists that requires only the UTXOs and not the entire blockchain history. This method uses an authenticated data structure that stores the UTXOs in a DHT.
