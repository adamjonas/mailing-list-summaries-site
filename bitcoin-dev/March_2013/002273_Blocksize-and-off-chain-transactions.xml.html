In response to the controversy over increasing blocksize as a solution to increased transaction volume, Gavin Andresen argues that raising the block size limit is necessary to scale up the blockchain. He acknowledges the need for off-chain transactions for "instantly confirmed" transactions, but believes that discussion should be tabled until version 0.8.1 is released.
