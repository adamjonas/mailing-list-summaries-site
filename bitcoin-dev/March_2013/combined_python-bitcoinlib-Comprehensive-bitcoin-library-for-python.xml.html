Summary:

The python-bitcoinlib project is a user-friendly library designed for pynode that provides an easy object interface to all bitcoin core data structures including blocks, transactions and addresses. It also features a full transaction script engine and ECDSA verification via OpenSSL wrapper. The library offers an object interface to all known network messages as well as binary encoding/decoding for full bitcoin protocol interoperability. Additionally, it has full verification of main and testnet block chains via pynode and passes many of the tests shipped with the bitcoin reference client.Despite its impressive features, developer Jeff Garzik suggests that the library is only recommended for developers and not for highly secure production sites. He welcomes pull requests, comments, questions, and donations.
