  • Click here to read original discussion on the bitcoin-dev mailing list

Summary:

The latest update of Bitcoin-Qt/bitcoind version 0.8.1 is now available for download from SourceForge. This release is a maintenance update that includes a new network rule to prevent chain-forking incompatibility with versions 0.7.2 and earlier. The source code for this update is available in a 0.8.1 tree that was branched directly from the 0.8.0 release at GitHub. Gavin Andresen, the Chief Scientist at the Bitcoin Foundation, made the announcement.
