Summary:

In the Bitcoin world, once a supermajority of mining reaches block version 2 (95%), blocks with nVersion==1 will be rejected. This is expected to happen in the next week. The Version 2 block specification is available on the Bitcoin wiki page for BIP 0034. You can monitor the status of this event at http://blockorigin.pfoe.be/top.php. Users should not be affected by this change, but some older miners may produce newly-invalid blocks (v1) that get ignored. Upgrading to a recent bitcoind (0.7 or later) is the easy solution for miners. If any miner needs help upgrading to v2, they can reach out to #bitcoin-dev or bitcoin-development at lists.sourceforge.net.
