Summary:

A proposal to improve privacy in the Bitcoin P2P network has been put forward by Bradley Denby. The Dandelion protocol aims to provide greater anonymity for users by ensuring that transactions are routed through a number of different nodes before being broadcast to the wider network. While the idea is yet to be implemented, the BIP (Bitcoin Improvement Proposal) has been shared with the community for consideration. However, Pieter Wuille, a prominent developer at Blockstream, has raised several issues with the current BIP, including the need to describe intended node behavior more clearly and uncertainty over how the protocol will interact with other elements of the Bitcoin ecosystem.
