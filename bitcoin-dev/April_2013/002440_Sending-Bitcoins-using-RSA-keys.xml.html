Summary:

There are two different types of digital payments, with bitcoin using ECDSA and GPG while Payswarm/WebID mostly use RSA. It is possible to bring these two worlds together to enable bitcoins to be sent over webid or payswarm. The problem arises when Alice and Bob have RSA key pairs but no public bitcoin addresses, and Alice wants to send 1 BTC to Bob. To solve this issue, Alice takes Bob's WebID and encrypts it with her private key to create entropy. She then uses that message as the seed to produce a Bitcoin address with an ECDSA key pair. Alice sends coins to this address and then encrypts the seed again with Bob's public key. Bob decrypts the seed using his private key and can now use the seed to recreate the wallet and spend the coins. This unites the web paradigm and crypto currency paradigm into one potentially giant ecosystem.
