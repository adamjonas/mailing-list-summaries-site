https://lists.linuxfoundation.org/pipermail/bitcoin-dev/2021-April/018889.html When creating a UTXO (unspent transaction output) in Taproot, all spends are indistinguishable as long as the keys used are not reused. This means that if the keys are not reused, it is impossible to tell which spending conditions were actually required for the spend. However, if keys are reused, it may have security and privacy implications. It is unclear whether reusing keys in Taproot has different implications than with Pay-to-PubKey-Hash.
