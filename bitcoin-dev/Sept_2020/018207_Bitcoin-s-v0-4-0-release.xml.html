Summary:

The team behind bitcoin-s has recently released version 0.4.0 of the software, which includes several major features such as a wallet overhaul to support all common script types, SQLite and Postgres database support for all projects, improved robustness of the Neutrino node, BIP340 Schnorr Signatures implemented in Java (Bouncy Castle), wallet rescans with block filters, and a test kit that contains all new features so developers can re-use the code to test their bitcoin applications. The release notes can be found on their GitHub page or in the 'Getting Started' section on their website.
