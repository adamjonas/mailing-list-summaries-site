Summary:

Pieter spent some time working on the optimal parameter selection for the Golomb Coded Sets proposed in BIP158. The details of the work can be found on his GitHub page. He found that if an FP rate of exactly 1 in 2^20 is desired, the Rice parameter should be 19 instead of 20. If a different FP rate is preferred, such as 1 in a 1.4971*2^B, then M=784931 B=19 or M=1569861 B=20 would be good choices.
