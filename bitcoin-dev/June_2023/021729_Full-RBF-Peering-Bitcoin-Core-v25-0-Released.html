Summary:

Bitcoin Core version v25.0 is now available for download from bitcoincore.org, or by cloning the git repository from petertodd/bitcoin/full-rbf-v25.0. This version includes Antoine Riard's full-rbf peering code along with some minor updates. Full-rbf nodes can advertise a FULL_RBF service bit when mempoolfullrbf=1 is set, and can connect to four additional FULL_RBF peers, ensuring reliable propagation of full-rbf replacements. While not everyone needs to run this version, more people running it would be helpful. A blog post by Peter Todd explains why one should run mempoolfullrbf. Additionally, there are hats available for those interested.
