Summary:

A Bitcoin developer has shared a proof of concept for payjoin(p2ep) that does not require any personal server, addressing concerns about its adoption. While stowaway, used by samourai, uses servers, the developer's proposal only needs common nostr relays between sender and recipient. The repository for this proof of concept can be found on GitLab, and a demo video is available on YouTube.
