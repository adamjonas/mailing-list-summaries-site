Summary:

Tony, a Brazilian software engineer, has developed a free software crowdfunding platform called FreedomSponsors. The platform allows individuals to place money bounties on issues of open source projects and multiple people can contribute to the same offer if they have similar needs. Payments are made directly to developers upon issue resolution. Presently, the only payment method supported is Paypal, but Tony is working on enabling bitcoin as both a currency and payment option. The platform is open source on Github and Tony welcomes feedback from users.
