Summary:

Mark Friedenbach, a developer, faced difficulties while setting up and maintaining his own gitian build system. In order to simplify the process, he documented the necessary steps into a makefile. This makes doing a gitian build as simple as running "cd contrib/vagrant && make" on any platform with the required dependencies installed. Mark has submitted these scripts as pull-request #1597 on Github and hopes that others will find it useful.In an email conversation between Mark Friedenbach and Wladimir, they discuss the development of Virtualbox and its importance in gitian builds. According to Wladimir, Virtualbox has fewer CPU requirements and is usable on older CPUs when compared to KVM, which is slower without hardware virtualization. Additionally, KVM and Virtualbox do not work well together.The email thread also features a link to an exclusive live event that discusses today's security and threat landscape, including endpoint security, mobile security, and the latest in malware threats. It also provides links to the Bitcoin-development mailing list.
