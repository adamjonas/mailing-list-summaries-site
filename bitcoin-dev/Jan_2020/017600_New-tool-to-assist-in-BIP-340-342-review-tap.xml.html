Summary:

The Bitcoin Script Debugger (btcdeb) has released a new experimental branch called "taproot" which includes a new command line tool named "tap". This tool allows users to compose taproot and/or tapscript outputs for funding and spending. The taproot build is based on the WIP taproot pull request to Bitcoin Core. A detailed tutorial on how to use this tool can be found on their Github page. Feedback on the new tool is appreciated.
