Summary:

Bitcoin Knots has released a new version, 0.19.0.1.knots20200104, which includes several new features, bug fixes, and performance improvements. This version enables BIP157 (Neutrino) to serve compact block filters to peers, expands PSBT support to include new fields and easier usage from the GUI for watch-only wallets, and adds an experimental feature to the Overview tab that allows users to hide private information. Users are encouraged to report any bugs by using the issue tracker on GitHub and subscribe to receive security and update notifications. Suggestions for improving this feature can be made on the related Core PR. Additionally, the updated translations for the latest version have been included in the release notes and change log available on GitHub. The release notes also highlight other notable changes in the new version. The latest version is intended to improve user experience and address issues from previous versions.
