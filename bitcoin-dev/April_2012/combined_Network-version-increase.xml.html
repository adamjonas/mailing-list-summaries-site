Summary:

On April 2, 2012, an email thread was initiated by Jeff Garzik to discuss the numbering scheme currently used in Bitcoin, which is 60000. Pieter Wuille suggested two options - either increase the number by one for every change or perform a "cleanup" and start incrementing from 100000. Jeff Garzik preferred having 100000 as the baseline frozen protocol. Wladimir supported the idea of increasing the number by one for now.Recently, Mike Hearn submitted a pull request to add a pong message in reply to a ping, which requires an upgrade of the network protocol version number. The version number has been independent from the reference client since BIP14, and the current value is 60000. Pieter Wuille seeks opinions on whether to simply increase the number by one for every change or do a "cleanup" and start incrementing from 100000.
