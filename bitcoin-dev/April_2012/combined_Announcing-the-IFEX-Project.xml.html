Summary:

XChange is a project that aims to provide a unified API for applications to access financial exchanges, currently supporting Bitcoin exchanges like MtGox and Intersango with plans to expand to others. The project is still in early development stages but could be integrated into the MultiBit Bitcoin client in the future. The XChange team faces challenges as each exchange uses its own data model and protocol, making it helpful to provide a reference implementation of how an exchange should publish its data.The Internet Financial EXchange (IFEX) Project seeks to enhance interoperability between financial settlement systems for the internet community, including conventional and digital currencies, alternative financial communities, and financial service providers. They offer the latest version of the IIBAN Proposal (v1) for financial endpoint identification on their website, which provides users with a democratically allocated 13-character identifier compatible with conventional banking infrastructure in Europe and other countries. IFEX's proposed infrastructure differs from existing projects as it is not a currency or settlement-network but rather a mechanism for bridging them, offering a broader and more inclusive scope than existing vendor-specific APIs and conventional finance industry networking protocols. The IFEX Protocol is a work in progress that aims to provide a standard protocol for transaction and settlement path negotiation with arbitrary financial instruments, currencies or assets, enabling better connectivity, lower settlement fees, real-time redundant financial routing, and arbitrary instrument/currency/asset handling. The goal is to move towards an open-source implementation of the IFEX protocol that interoperates with major and emerging settlement networks for the benefit of all parts of the community.
