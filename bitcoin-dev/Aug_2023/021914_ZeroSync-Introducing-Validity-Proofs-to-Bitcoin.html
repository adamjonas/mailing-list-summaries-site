Summary:

There is a general-purpose zkVM implementation available for the RISC-V instruction set. This implementation can be found at https://www.risczero.com/. It is worth noting that Bitcoin Core can be compiled for RISC-V, and with the presence of RiscZero, it is now possible to prove execution traces of a RISC-V VM. Consequently, any argument against using RISC-V for Bitcoin Core no longer holds true. With this implementation, the exact consensus rules can be applied and verified. This information was shared by Peter Todd in an email conversation with @blk014.
