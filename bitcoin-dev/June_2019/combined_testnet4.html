Summary:

Bitcoin developer Emil Engler has suggested resetting the testnet blockchain with a new genesis block to reduce the time it takes to sync and the size of the blockchain. However, Bryan Bishop, a Bitcoin Core developer, doubts this will happen and recommends using Signet instead, which provides an alternative for configuring multiple separate private and public testing networks. Testnet is an important tool for developers to test new features and upgrades before they are implemented on the mainnet, but due to the nature of testnet coins having no value, the blockchain could be spammed after a reset until it reaches a certain size. Peter Todd argues that the size of testnet itself is an important test and has suggested in the past that testnet should be made larger than mainnet. Emil Engler suggests that there could be a better alternative to testnet, but also recognizes that it is still a useful tool and should not be ignored.
