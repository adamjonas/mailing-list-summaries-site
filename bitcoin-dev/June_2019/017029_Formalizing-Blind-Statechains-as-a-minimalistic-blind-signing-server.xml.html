Summary:

The conversation between Ruben Somsen and ZmnSCPxj revolves around Statechains, which is a system that allows for off-chain transfer of UTXO ownership. They discuss the "Stale Factory" and "Broken Factory" problems with channel factories, and how Statechains can address these issues by using `SIGHASH_NOINPUT`. Additionally, they consider privacy concerns and decide that combining Statechains via MuSig is best. While scripting would allow for programmable escrow, Ruben prefers to keep the system script-free. ZmnSCPxj notes that any updateable off-chain cryptocurrency system can accomplish what Statechains does, and suggests presenting further transactions to the system in order to append them to existing unilateral-case transactions and cut-through on the next update.
