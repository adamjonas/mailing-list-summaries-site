Summary:

Bitcoin Knots has made available the second release candidate for its new version, 0.14.0. This release includes new features, bugfixes, and performance improvements. The preliminary release notes can be found on their GitHub page, along with the source code under the signed tag. The binaries can be downloaded from the Bitcoin Knots website, and users are advised to verify the PGP signature of all downloads. This release candidate is a test version, and if no critical problems are found, it will be tagged as the final 0.14.0 release. Bugs can be reported using the issue tracker on GitHub.
