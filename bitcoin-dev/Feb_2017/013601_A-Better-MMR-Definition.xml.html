Summary:

Chris Priest and Peter Todd are discussing the definition of a prunable MMR for use in Bitcoin. The improved definition commits to the number of items in the tree implicitly, allowing for an obvious max-log2(n)-sized proof-of-tree-size by following the right-most nodes. This MMR could be used in TXO commitments, making for a better overall TXO commitment.
