Summary:

Peter Todd has made available a full-RBF patched branch of Bitcoin Core v0.14.0rc1 on his GitHub account. The new version uses the nRelevantServices machinery to perform preferential peering and requires only a few lines of code changes compared to the original version. The relevant services machinery is less aggressive in terms of connecting to full-RBF peers than the custom code used in earlier versions. This ensures that RBF peers can remain connected to each other, making it easier to maintain the patched branch every time there is a new upstream version released.
