Summary:

BitcoinMechanic has announced a Taproot activation meeting on IRC to be held on Tuesday, April 13th at 19:00 UTC. The purpose of the meeting is to ratify the Taproot activation plan discussed in the previous meeting on March 16th, also known as 2021-03 Plan Y. The focus will be on reaching a consensus on BIP8 and the remaining parameters. While there was no agreement on the LOT parameter, there seems to be a consensus on proceeding safely with LOT=True.The miners will have 18 months to signal and accelerate activation; otherwise, Taproot will activate regardless. With the majority of the economy running it, this guarantees eventual lock-in of taproot with the smallest chance of a chain split. Michael Folkson has encouraged other reviewers to review Core PR #21377 and recommends any alternative release be fully compatible with the activation parameters in Core if that PR is merged into Core. Bitcoin Mechanic has set up the meeting to discuss an alternative release to Core with Taproot activation code. Discussions will include deciding when Core should no longer be a consideration.Jeremy Rubin points out that there is a middle road to consider. If Core chooses not to release activation parameters, then #21377 can still be used to issue a community release safely. This would avoid the false dichotomy between an ST released by Core and a BIP8 UASF.Michael Folkson's contact details are also provided for anyone interested in attending the meeting. The channel remains open for ongoing discussions 24/7, and there is a web chat client available for use. The upcoming meeting promises to be a significant event in the development of Taproot activation, providing a platform for discussion and decision-making among key stakeholders in the Bitcoin community. Interested parties can refer to the summary of the previous meeting via the provided link.
