Summary:

In this conversation, Ruben proposes a mechanism for blind merged mining with a native token called "spacecoin." This mechanism avoids the need for a specific soft fork and limits burn outputs to one per block. The perpetual one-way peg allows for fair spacecoin creation by burning BTC, avoiding creating bad speculative altcoin incentives. The design only lets BMM block creators burn BTC, limiting potential burn outputs. Spacechains can be created inside the initial spacechain at no additional storage cost to Bitcoin. Ruben recommends checking out the links in his prior post to learn more, including a video.
