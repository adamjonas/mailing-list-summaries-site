Christopher Gilliard, a member of the cryptocurrency community, has created a new Bitcoin Improvement Proposal (BIP) that defines a decentralized naming protocol. The BIP maps names to onion addresses instead of IPs and is integrated into other systems proposed by Gilliard. The proposal can be found on GitHub where Gilliard invites feedback, questions, or comments from others. One respondent to the mailing list noted that the protocol was similar to Namecoin but without the alt. The proposed protocol appears to be similar to Namecoin, which is a decentralized naming system built on top of the Bitcoin blockchain. It remains to be seen how this new proposal will differ from existing systems like Namecoin and whether it will gain traction in the cryptocurrency community.
