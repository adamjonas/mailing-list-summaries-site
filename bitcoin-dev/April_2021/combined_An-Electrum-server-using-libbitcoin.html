Summary:

Ivan has developed an Electrum server implementation called Obelisk, which uses zmq and libbitcoin as its backend. The project aims to allow the use of the Electrum wallet with libbitcoin servers directly, without the need for a protocol/server between a client and daemon. Ivan plans to keep track of new Electrum/ElectrumX protocols and implement them in Obelisk as they come.Although merging Obelisk into the current Electrum git seems relatively simple, it would be a breaking change that invalidates all old servers. Despite this, Ivan believes that querying a libbitcoin server directly is not a bad idea. He welcomes feedback and review for Obelisk, which is available on GitHub.The implementation of Obelisk consists of around 1000 lines of Python 3 code with no external dependencies besides pyzmq.
