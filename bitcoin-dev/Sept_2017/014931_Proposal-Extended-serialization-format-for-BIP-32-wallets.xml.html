Summary:

Pavol Rusnak, the CTO of SatoshiLabs, has proposed a new idea to change bip32 version bytes for SegWit. He shared his proposal on GitHub, which can be found in the SL repo. The proposal aims to extend BIP32 and improve it further. Pavol encourages people to comment on his proposal and welcomes any feedback. He expresses his desire to have his proposal added to the BIP repo if there is enough interest.
