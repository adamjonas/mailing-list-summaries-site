Summary:

A proposal has been made to add a new command, 'mempool', to the satoshi client v0.7. This change would allow simplified payment verification (SPV) clients access to the memory pool and permit diagnostics and miner download. The "getdata" behavior would also be extended to include any memory pool transaction. This would relax a previous restriction on the "getdata" behavior. While this proposal did not require a Bitcoin Improvement Proposal (BIP), Jeff Garzik offered to write one if needed.
