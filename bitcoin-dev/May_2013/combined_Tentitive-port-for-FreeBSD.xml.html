Summary:

On May 25, 2013, Robert Backhaus sent an email to his colleagues containing a link to the FreeBSD build system 'port' for comments. The Makefile applies user requests for GUI/QR/UPNP and includes five patches in the files directory. Two of these patches add boost-crypto to LIBS, while another patch uses external leveldb. Two minor fixes are also included. Robert Backhaus has branches ready for pullreqs on those minor fixes and would double-check them before creating the pull requests that evening. Luke commented on the email thread encouraging others to help test and use the pull request for system LevelDB support. He urged caution when tying the port to the specific LevelDB version bundled to avoid any risk of unexpected hardforking bugs or fixes in upstream LevelDB.Robert Backhaus is planning to get the system committed once version 0.8.2 is released. The major change being the use of an external port for leveldb. The files directory contains five patches, including two that add boost-crypto to LIBS and one that patches the build to use the external leveldb. Two minor fixes are also included. Branches are available for pullreqs on those minor fixes. Interested parties can find the files at https://redports.org/browser/robbak/net-p2p/bitcoin. Any comments are welcome.
