Summary:

Bitcoin-Qt version 0.8.2 release candidate 1 has been made available on SourceForge, which focuses on maintenance and bug fixes along with a few new minor features. The default fee for low-priority transactions has been lowered from 0.0005 BTC to 0.0001 BTC and payments of 0.543 times the minimum relay fee are now considered non-standard. The changes include a new icon and splash screen, improved synchronization process reporting and the removal of hardcoded fee recommendations. Additionally, there are significant changes to the networking code that reduce latency and memory consumption.
