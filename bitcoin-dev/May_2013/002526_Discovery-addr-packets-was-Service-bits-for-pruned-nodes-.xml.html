Summary:

In an email thread discussing the value of D (difficulty) in determining the expected return in Bitcoins from the block reward had the hasher solved valid blocks, Peter Todd suggests using the txid:vout pair of a txout that can only be spent once as a non-reusable proof-of-work for transferable assets on the blockchain. This method would require less communication than Bitcoin, but would take away from the overall security of the network and make it easier to rent hashing power. Todd notes that the final proof is more compact than a fidelity bond proof.
