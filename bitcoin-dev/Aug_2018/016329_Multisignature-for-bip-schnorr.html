A developer named nakagat posted a multisignature procedure using bip-schnorr to the bitcoin-dev mailing list on August 13, 2018. The post requested feedback on the implementation. The original implementation can be found on GitHub, along with information about Multisignatures and Threshold Signatures using bip-schnorr. While the method is cool, there are a lot of online steps and it's not a threshold system. However, using a shamir scheme solves these issues and isn't subject to birthday attacks. A link to a Medium article explaining this solution was provided.
