Summary:

The Chia Network has developed a new library for BLS signatures that allows for non-interactive aggregation. This means that the library can be used to perform aggregation without requiring any interaction between parties, making it more efficient than Schnorr signatures. The library is fully functional and based on a construction using musig. The Chia Network is inviting feedback, discussion, and usage of the library. The library can be accessed at https://github.com/Chia-Network/bls-signatures.
