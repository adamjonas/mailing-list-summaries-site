Summary:

On August 3, 2018, Bram Cohen, the creator of BitTorrent, posted on the bitcoin-dev mailing list about a new development in BLS signatures. These signatures have the same aggregation feature as Schnorr signatures but can be done non-interactively with better aggregation at slower speeds. A first draft library for BLS signatures was released, based on a musig construction, which is fully functional. The Chia Network has developed this library for BLS signatures that allows for non-interactive aggregation making it more efficient than Schnorr signatures. The library is based on a construction using musig and invites feedback, discussion, and usage from the community. The announcement was made on the bitcoin-dev mailing list and a comparison to Shigeo's implementation was also mentioned in the thread. The library can be accessed at https://github.com/Chia-Network/bls-signatures under the Chia-Network organization.
