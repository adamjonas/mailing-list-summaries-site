Summary:

Lautaro Dragan, the Tech Lead of Po.et, has reached out to bitcoin-dev for guidance on using colored coins or storing data on the Bitcoin blockchain with a prefix. He has read that protocols that wish to claim OP_RETURN prefixes should use the standard Bitcoin Improvement Proposals (BIP) process. However, he has not been able to find any existing BIP claiming an OP_RETURN prefix. Lautaro is seeking advice on whether to use the BIP process to claim a prefix. If Lautaro decides to use colored coins, he needs to collaborate with the authors of BIP 160 or write a new BIP if BIP 160 is insufficient for some reason. On the other hand, if he decides to store data on the blockchain, he should not do it at all. BIPs need to specify an actual protocol, not just claim a prefix.
