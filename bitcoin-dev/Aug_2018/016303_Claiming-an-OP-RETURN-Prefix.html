Summary:

In a discussion on the Bitcoin development mailing list, Jude Nelson asked whether miners can identify transactions from certain software and censor them. Christopher Allen responded that in the case of InterPlanetary File System (IPFS), where hashes have to be inspected to know which content to censor, the latency required would likely impact the miner's ability to succeed in the block. Therefore, as long as all hashes look the same, multiple content addressable schemes that use hashes will need to be censored as a whole or not at all.
