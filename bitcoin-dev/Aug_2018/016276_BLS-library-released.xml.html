Summary:

On August 3, 2018, Bram Cohen posted on the bitcoin-dev mailing list about a new development in BLS signatures. These signatures have the same aggregation feature as Schnorr signatures, but with the added benefit of being able to be done non-interactively, albeit at a slower pace. A first draft library for BLS signatures was released, based on a musig construction. The library is fully functional and feedback, discussion, and usage are all welcome. The code can be found on GitHub at https://github.com/Chia-Network/bls-signatures.
