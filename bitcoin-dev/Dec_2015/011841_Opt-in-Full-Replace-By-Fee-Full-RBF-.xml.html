Summary:

A discussion over the use of Sequence for opting in took place on Bitcoin-dev mailing list. The CTxOut data structure does not contain any extra fields to be used for opt-in per output. A suggestion was made to use the same scheme as sighash_single, but it poses a privacy issue as it reveals change address information. Pre-committing via a hash and revealing it later is complex and not easily feasible with the current tx data structure.
