On December 20, 2015, Jeff Garzik via bitcoin-dev expressed concerns over the reversal of the useful minimization attribute of HD wallets, which is the "just backup the seed." He suggested that it would be nice if the bip37 filter matching algorithm was extended to serve up the proof. This would preserve the "just backup the seed" functionality if server-based wallets did the same.
