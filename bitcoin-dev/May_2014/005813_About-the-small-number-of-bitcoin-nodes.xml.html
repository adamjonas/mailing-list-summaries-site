Summary:

In a message related to the number of bitcoin full nodes on a mailing list, a user who owns two Bitcoin Nodes suggested some ideas. These include allowing users to view the bandwidth used by Bitcoin Core, educating users about the correct setup of a bitcoin node, creating a default config file with a random RCP password and username for bitcoind and Bitcoin Core, adding Bitcoin Core and bitcoind to Linux repos, creating a graphical interface for bitcoind on Linux servers, splitting Bitcoin Wallet from Bitcoin Node, informing users if 8333 port is closed, and keeping connections if bitcoind is restarted. The user believes these ideas would contribute to enhancing the network.
