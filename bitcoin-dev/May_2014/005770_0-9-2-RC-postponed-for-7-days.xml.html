Summary:

The release candidate for version 0.9.2 of Bitcoin was scheduled to be released on May 13th, but due to the Bitcoin 2014 Amsterdam conference, it has been postponed for seven days. During this time, a string freeze is in effect and contributions for translations are welcome. Additionally, nightly builds for both the master and 0.9.2 branches can now be found at http://nightly.bitcoin.it. The 0.9.2 branch will soon become the official release, while the master branch is accepting changes that may not be safe for a near-term release. Finally, there are details available about deterministic reproducibility of the unofficial nightly builds. Warren Togami is responsible for this update.
