Summary:

On April 30th, 2014, Kristov Atlas reported an issue with a binary built by Wladimir where the GUI segfaulted after he chose the default datadir. Wladimir fixed the issue and created a new test build which is available for download from visucore.com. The dev binary runs smoothly in Tails v1.0, as confirmed by Kristov Atlas. However, it is not clear whether this will make the usual Linux binary compatible or if there will be a special binary just for systems running the older Qt.
