The conversation was centered around the unit test framework for Bitcoin. Matt Corallo mentioned that they already had the boost testing framework implemented, but it lacked tests. Rick Wesson suggested using Google Test and provided a link to it. John Smith pointed out that the current unit test framework did not build by default and provided two ways to fix it. The first way was to add #define BOOST_TEST_DYN_LINK to the top of test_bitcoin.cpp. The second way was to link boost_unit_test_framework statically using the code provided. Matt suggested checking out pull requests 363 and 442 for more information.
