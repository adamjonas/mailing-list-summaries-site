The Mac OSX version of the bitcoin client has been posted on SourceForge. The updated checksums for each file have also been provided and stored as SHA1SUMS.asc on SF. The files available for download include bitcoin-0.3.24-linux.tar.gz, bitcoin-0.3.24-macosx.zip, bitcoin-0.3.24-src.tar.gz, bitcoin-0.3.24-win32-setup.exe, and bitcoin-0.3.24-win32.zip. The PGP signature is also included in the message.
