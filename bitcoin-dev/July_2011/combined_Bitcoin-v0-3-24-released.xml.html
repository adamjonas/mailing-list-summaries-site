Summary:

The latest version, v0.3.24 of Bitcoin is now available for download from SourceForge. This release mainly focuses on bug fixes as the existing clients were facing issues with block downloads and network connection loss. Although the team intended to have wallet encryption ready for release, it will be included in the next version, v0.4. Notable changes in v0.3.24 include default DNS seeding, UPnP enabled by default in the GUI client, initial unit testing framework, and internal wallet code cleanup. The Github page has a full changelog and PGP signatures for verification.In addition, the Mac OSX version of the bitcoin client is now available on SourceForge. The updated checksums for each file have been provided and stored as SHA1SUMS.asc on SF. The message includes the files available for download such as bitcoin-0.3.24-linux.tar.gz, bitcoin-0.3.24-macosx.zip, bitcoin-0.3.24-src.tar.gz, bitcoin-0.3.24-win32-setup.exe, and bitcoin-0.3.24-win32.zip.
