Summary:

In an email conversation on March 27, 2014, Tamas Blummer proposed a new method for identifying cryptocurrencies. Instead of creating a directory of magic numbers to represent each currency, Blummer suggested using the hash of the currency's name to avoid collisions and eliminate the need for a central directory. Jeff Garzik, a Bitcoin core developer and open source evangelist, responded positively to Blummer's suggestion, giving it his endorsement with a "+1 good idea."
