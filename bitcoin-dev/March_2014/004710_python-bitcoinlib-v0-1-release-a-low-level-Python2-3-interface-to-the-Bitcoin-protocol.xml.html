The email thread reveals a discussion between Jeff Garzik, a Bitcoin core developer and open source evangelist, and Peter Todd, discussing the python-bitcoinlib dependency. Todd states that the ngccbase Colored Coin client has added his fork of python-bitcoinlib as a dependency. He further mentions the existence of a rudimentary python-bitcoinlib package in archlinux. Todd attributes the bulk of the code structure of his fork to Jeff Garzik's implementation, while also clarifying that he has made some changes to "pythonize" the code. In turn, Garzik credits the project's inception to ArtForz, who started the project with his half-a-node. Todd provides a link to Garzik's original implementation on Github.
