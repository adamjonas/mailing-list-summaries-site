Summary:

Bitcoin developers have discovered that secure two-way pegging between side chains is possible with some changes to bitcoin. This allows interoperability, as users can move bitcoin into and out of a side chain. The side chain may have different parameters or experimental features subject to a minimum compatibility level that enables the production of an SPV proof of a given form. Greg Maxwell proposed a compact way to do two-way pegging using SPV proofs last year, which also provides a security firewall. Bitcoin has no mining reward on the beta chain network but allows users to mint coins at no mining cost by providing an SPV proof that the coin has been suspended in bitcoin.
