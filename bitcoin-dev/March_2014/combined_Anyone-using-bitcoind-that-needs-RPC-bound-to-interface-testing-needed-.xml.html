Summary:

Bitcoin Core developer Wladimir van der Laan has developed a new patch that enables users to bind the RPC port to specific addresses or multiple address/port pairs. The patch is available on Github, and Wladimir is encouraging users to test it before merging. He is welcoming test results from normal setups as well as unusual corner cases, such as IPv4+IPv6 and VPNs. This development is aimed at increasing security for Bitcoin users by allowing them to bind their RPC ports to specific addresses or ports.
