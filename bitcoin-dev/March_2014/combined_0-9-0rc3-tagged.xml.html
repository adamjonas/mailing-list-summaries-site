Summary:

Binaries for the Bitcoin 0.9.0rc3 version can now be found on the official website. The community is encouraged to participate in testing to ensure that the release is stable and free of bugs. More 'gitian builders' are needed for the final 0.9.0 release, as only two builders have built the rc3 binaries so far. Those running Linux or OSX are invited to help by building dependencies on their virtual machines.The new version, 0.9.0rc3, has been tagged, and users are advised to start their gitian builds. This version includes changes to the mining code fee policy, which now matches the relay fee policy. Additionally, a rare crash in the wallet code has been fixed, and there have been some small changes to the build system, documentation, and GUI.No message regarding this update has been posted on the mailing list yet.
