Summary:

In a discussion about identifying coins, Allen suggests using the unique magic numbers assigned to different cryptocurrencies like Bitcoin and Litecoin. However, Jeff Garzik proposes an alternative solution - using a hash of the coin's name instead of a directory of magic numbers. Tamas Blummer also chimes in, suggesting that this approach would minimize the chance of collision while eliminating the need for a central directory. The idea is well-received, with Jeff giving it a +1 and introducing himself as a Bitcoin core developer and open source evangelist.
