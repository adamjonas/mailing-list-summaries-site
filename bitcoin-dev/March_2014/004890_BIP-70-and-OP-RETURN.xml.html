Summary:

The Output message allows for the specification of any standard TxOut script, including the ability to specify OP_RETURN outputs in BIP 70 payment requests. In the event that the creator of a payment request wished to include a small amount of data as an OP_RETURN output, they would need to specify this information. It is recommended to support online privacy by using email encryption whenever possible, and there is a resource available to learn how to do so. Additionally, attachments such as a PGP key and digital signature can be used for added security measures.
