Summary:

Bitcoin Core has released a new minor version, 0.10.3, with security fixes and translation updates. Users are advised to upgrade their software from the Bitcoin Core website or report bugs through GitHub. The upgrade also includes changes to fix a buffer overflow in the bundled miniupnpc, disable upnp by default, test for lowS signatures before relaying and increase the minimum relay fee default from 0.00001 to 0.00005 due to current transaction flooding causing memory usage on nodes. Details of the other changes can be found in the change log.
