Van der Laan responded that most of the investment in the network is not in the Bitcoin Core software, but in things building on top of it. The assumption being made is that nothing really stupid will happen to the network/protocol and that it will not go down. The Bitcoin Core team recently released an API that makes implementing a full node practical, a reasonably usable Bitcoin Core GUI for non-technical people, and a reasonably safe wallet through the Bitcoin Core Daemon.
