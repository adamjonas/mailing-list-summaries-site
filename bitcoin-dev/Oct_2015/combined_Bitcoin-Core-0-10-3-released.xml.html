Summary:

Bitcoin Core, the open-source software that powers the Bitcoin network has released a new minor version 0.10.3. The new release includes security fixes and translation updates and users are advised to upgrade their software from the Bitcoin Core website or report bugs through GitHub. The new version has made several changes such as fixing a buffer overflow in the bundled miniupnpc, disabling upnp by default, testing for lowS signatures before relaying and increasing the minimum relay fee default from 0.00001 to 0.00005 due to current transaction flooding causing memory usage on nodes. The change log provides details of other changes made in this release. This update is crucial as it addresses some of the major security concerns regarding the use of Bitcoin. Since Bitcoin is an online currency that is based on cryptography, any vulnerability in the software can lead to potential security breaches. Therefore, it is recommended that users upgrade their software to the latest version to ensure the security of their transactions.
