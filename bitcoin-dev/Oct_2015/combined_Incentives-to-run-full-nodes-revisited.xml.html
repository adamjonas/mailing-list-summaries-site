Summary:

The developer has updated a previous post about a protocol concept designed to enable decentralization and expansion of a giving economy. The original post had fractured links, but the new implementation can now be found at http://abis.io. This concept is feasible in BTC and has already been implemented in the BCN-GUI wallet. Interested parties can find the developer's blog at https://bytecoin.org/news/bytecoin-wallet-1.0.8-release-introduces-micro-donations/, which can also be accessed through http://abis.io. Comments on the implementation are invited by the developer. Additionally, an interesting link related to this topic can be found at https://odinn.cyberguerrilla.org/index.php/2015/10/03/greater-giving-potential/.
