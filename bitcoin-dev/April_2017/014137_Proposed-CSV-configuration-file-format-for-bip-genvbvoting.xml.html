Summary:

Sancho has shared a proposed CSV-based file format for fork deployment data, which is tentatively named forks.csv. This format will be used in the reference implementation of bip-genvbvoting that is still under development, and other BIP9 improvement proposals are also encouraged to use this format. Sancho is open to discussing extensions of this format for supporting flag days or direct-to-activation transitions. The documentation for this proposal can be found at https://raw.githubusercontent.com/sanch0panza/bitcoin/genvbvoting-bu-dev/doc/genvbvoting.md.
