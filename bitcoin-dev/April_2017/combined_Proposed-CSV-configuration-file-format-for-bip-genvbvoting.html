Summary:

Sancho has proposed a CSV-based file format named forks.csv for fork deployment data. This format will be utilized in the reference implementation of bip-genvbvoting, which is still under development. Other BIP9 improvement proposals are also encouraged to use this format. Sancho is open to discussing extensions of this format for supporting flag days or direct-to-activation transitions. In order to learn more about this proposal, the documentation can be found at https://raw.githubusercontent.com/sanch0panza/bitcoin/genvbvoting-bu-dev/doc/genvbvoting.md.
