Summary:

Bitcoin v0.3.23rc1 releases for Win32, Linux and source have been uploaded to SourceForge in response to an influx of new bitcoin users. P2P connect-to-node logic has been changed to reduce timeout and hunt for useful P2P connections due to new users not permitting incoming connections. Transaction fees have been reduced to 0.0005 for new transactions, while client will relay transactions with fees as low as 0.0001 BTC. The Git changelog lists a number of translations that have been updated, compliance fixes for ignoring non-internet-routable host, adding common temp files to .gitignore, and demystifying some magic numbers. There are also bug fixes for CPU usage when using -nolisten and having no connections, creating keypool in LoadWallet(), updating transactions already in the wallet when rescanning, fixing small change outputs, bumping version to 0.3.23, reducing minimum TX fee for new transactions, and adding minimal release process docs.
