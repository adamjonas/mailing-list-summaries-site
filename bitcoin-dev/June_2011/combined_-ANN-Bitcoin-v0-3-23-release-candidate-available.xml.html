Summary:

The latest version of Bitcoin, v0.3.23rc1, has been released for Win32, Linux and source. This update is focused on addressing the increasing number of new bitcoin users. The P2P connect-to-node logic has been changed to reduce timeout, which will enable quicker hunting of useful P2P connections. Additionally, transaction fees have been reduced to 0.0005 for new transactions, while fees as low as 0.0001 BTC will still be supported.The release includes several other bugfixes and updates, such as updated translations for Chinese Simplified and Swedish languages, ignoring non-internet-routable hosts, adding common temp files to .gitignore, fixing CPU usage bugs and creating keypool in LoadWallet(). With these updates, Bitcoin aims to provide a smoother and more efficient user experience for its growing user base.
