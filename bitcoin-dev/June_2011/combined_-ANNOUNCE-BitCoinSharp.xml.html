Summary:

Nathan Baulch has announced the BitCoinSharp project, which is a complete port of BitCoinJ into C#. This will open up the world of Bitcoin to .NET developers everywhere. The code for the project can be browsed on http://code.google.com/p/bitcoinsharp/source/browse/. Visual Studio SLN files are provided to help get started and the PingService example is a good starting point. The announcement hopes that they have covered all the bases in terms of languages and platforms and everyone should be able to write interesting Bitcoin apps no matter their background or preferred toolchain. A request was made by Laszlo Hanyecz to include the recent fix to join a random #bitcoinXX channel on IRC instead of #bitcoin.
