Summary:

The email announces the release of BitCoinJ v0.2, which includes over 100 commits since the first release in March. The new version offers full support for block chain re-orgs and recognition of dead transactions that will never be included in the chain. Additionally, it features persistence of the block chain using multiple pluggable stores, including a BoundedOverheadBlockStore suitable for usage on mobile devices where low memory usage and instant startup time are requirements. Other updates include IRC, DNS, and seed list peer discovery, ASN.1 key export, many bug fixes, and minor API improvements. The release is the work of many individuals, including Andreas Schildbach, Miron Cuperman, Gary Rowe, Thilo Planz, Micheal Swiggs, Noa Resare, John Sample, and Xiaofeng Guo. The announcement was also sent to the bitcoin-development mailing list.
