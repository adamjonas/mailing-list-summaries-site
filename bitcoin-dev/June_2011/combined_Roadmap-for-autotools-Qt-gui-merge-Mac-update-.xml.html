Summary:

A developer named Mark has made some small changes to the bitcoin-qt client and autotools on Mac, which build and run without any issues. However, he needs more time to work on Autotools + qt. Mark will push his changes to Github for others to take a look at them. Meanwhile, Pieter Wuille, another developer, proposes merging autotools and qtgui on the Bitcoin-development mailing list. He also mentions that autotools require more testing on native windows and OSX platforms. Therefore, he requests anyone with these development environments and willingness to test it, to come forward.Overall, developers are discussing the possibility of merging autotools and qtgui, but they need to conduct more testing before doing so.
