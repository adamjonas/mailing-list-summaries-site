Summary:

Mark has spent time working with the bitcoin-qt client and autotools on Mac. He mentions that the qt client builds and runs with few small changes, and Autotools + qt will take a bit longer. Mark is planning to push the changes to Github so that anyone can work on it. In the email thread, Pieter Wuille is asking for help from developers who have development environments and willingness to test autotools on native windows and OSX platforms for Bitcoin-development mailing list. The consensus seems to be merging autotools and qtgui, but there are concerns about the lack of testing on native platforms. The email also includes information about EditLive Enterprise as the world's most technically advanced content authoring tool.
