Summary:

The development team has proposed the current git HEAD as version 0.3.23-rc1, pending approval from Gavin. The release is intended to be a fast, bugfix-only update with a focus on fixing the issue of connections taking forever, caused by a large portion of the network not accepting incoming TCP connections. The Git shortlog for 0.3.23-rc1 shows contributions from various developers including Dean Lee, Doug Huff, Han Lin Yap, Jeff Garzik, Matt Corallo, Nils Schneider, and Pieter Wuille. Some notable updates include Dean Lee's Chinese Simp translation, Doug Huff's fixes for rfc1918 and rfc3927 compliance, and Han Lin Yap's addition of Swedish translation and removal of an unused variable. Jeff Garzik made several contributions, such as bumping the version number to 0.3.23, reducing the minimum TX fee for new transactions, lowering the minimum relay TX fee, and adding minimal release process documentation. Matt Corallo fixed the CPU usage bug when using -nolisten and having no connections, while Nils Schneider created keypool in LoadWallet() and Pieter Wuille updated transactions already in the wallet when rescanning, among other improvements. Overall, the release focuses on bug fixes and performance improvements, and builds are ready to upload pending approval from Gavin.
