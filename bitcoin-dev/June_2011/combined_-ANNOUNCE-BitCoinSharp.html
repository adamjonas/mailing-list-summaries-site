Summary:

On June 27, 2011, Mike Hearn announced the BitCoinSharp project by Nathan Baulch via email. The project is a complete port of BitCoinJ into C#, which makes it possible for .NET developers to create Bitcoin apps using their preferred toolchain. The code can be browsed on Google Code and Visual Studio SLN files are provided to help get started. The PingService example is suggested as a good starting point for reading. Hearn hopes that this will enable everyone to write interesting Bitcoin apps regardless of background or platform. Additionally, the email includes a link to Splunk, which helps make sense of the valuable data generated in IT infrastructure. On June 28, 2011, Laszlo Hanyecz requested the inclusion of a recent fix to join a random #bitcoinXX channel on IRC instead of #bitcoin. The author expresses hope that the nearly all languages and platforms have been covered in the BitCoinSharp project, allowing developers from various backgrounds and toolchains to write interesting Bitcoin apps.
