Summary:

The writer proposes a draft BIP that defines a new transaction template that takes the next available transaction version number. This proposal is not expected to bring any consensus changes and is meant for the application layer of Bitcoin. The new transaction template aims to define a unique transaction structure that can be used as a cryptographic pseudonym. The writer hopes that the community will find this proposal useful and provide a careful review.The first BIP within the project is available on Github at https://github.com/veleslavs/bips/blob/bip_nym_tx/bip-nym_tx.mediawiki. The writer thanks the entire team that supported him in creating this proposal.
