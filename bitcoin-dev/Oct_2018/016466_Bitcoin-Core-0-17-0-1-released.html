Summary:

Bitcoin Core version 0.17.0.1 has been released and is available for download on its official website or through BitTorrent. The new major version release includes new features, various bug fixes, performance improvements, and updated translations. Users are advised to report bugs using the issue tracker at GitHub and subscribe to receive security and update notifications.The latest version has fixed an issue with OSX dmg generation that could cause Finder to crash on installation. This problem only affected macOS 10.12 to 10.14 and there are no significant changes for other operating systems. The release credits Jonas Schnelli, Pieter Wuille, Sjors Provoost, and Wladimir J. van der Laan for their contributions to this release. The announcement also includes PGP signature verification to ensure the authenticity of the message.
