Summary:

The writer has an idea related to Bitcoin's BIP 21, but notes that there is currently no Multisig request URI. The proposed idea involves a bitcoin-sigreq URI scheme, which would be used for a multisig request from a website to a user's local wallet. The URI scheme includes a raw transaction encoded by base64 as well as parameters for the public key for sign key pair and path for HD wallet position. The writer seeks feedback on the idea.
