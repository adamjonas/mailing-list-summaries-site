Summary:

Chris has suggested adding an optional note field to transactions involving Bitcoin. The purpose of this suggestion is to allow the receiver of the Bitcoin to know who sent the currency. Currently, there is no way for the recipient to know the identity of the sender unless they explicitly state it in the message accompanying the transaction. By adding a note field, this information can be included in a more organized and streamlined manner.
