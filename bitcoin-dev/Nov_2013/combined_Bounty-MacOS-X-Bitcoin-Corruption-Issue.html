Summary:

MacOS X users have been experiencing leveldb data corruption issues since the 0.8.x version of Bitcoin, with at least one mysterious corruption issue occurring after a clean shutdown of Bitcoin-Qt. While three commits have been made to fix the problem, there is still work to be done. In response, Gavin and Litecoin Dev have pledged 5 BTC and 200 LTC respectively to start a bounty for the fix. The thread has public addresses for Mac users to donate to increase the incentive to fix this issue sooner.For testing purposes, users are encouraged to use either Bitcoin git master or Bitcoin 0.8.5 OMG3, both containing all relevant leveldb fixes. Additionally, users are also urged to contribute detailed bug reports or links to more relevant background information pertaining to this corruption issue.
