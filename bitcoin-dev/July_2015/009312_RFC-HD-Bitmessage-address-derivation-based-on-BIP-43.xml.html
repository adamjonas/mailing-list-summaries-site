Summary:

The Monetas company has developed a Bitmessage address derivation method from an HD seed based on BIP-43. This method allows Bitmessage users to generate their addresses from a seed, providing eternal key backups. The same seed can also be used for a Bitcoin wallet and a Bitmessage client. Furthermore, this method enables future use cases where senders calculate Bitmessage addresses based on a recipient's extended public key and some other index value. Monetas is proposing this as a BIP per the BIP-43 recommendation in order to reserve a purpose code.
