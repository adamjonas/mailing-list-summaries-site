Summary:

Bitcore-wallet-server (BWS) is a facilitator for HD multisig wallets. Currently, BWS instances hold the set of extended public keys of the wallet's peers to derive addresses. The server does not have to derive keys itself beyond knowing that the addresses are part of the wallet. It is questioned whether the server could work as just a bucket of arbitrary addresses with clients deriving the addresses themselves.
