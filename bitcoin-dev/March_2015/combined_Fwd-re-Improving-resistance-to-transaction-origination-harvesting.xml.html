Summary:

Arne Bab and Justus Ranvier discussed a proposal for a bitcoin darknet (friend-to-friend) in an email exchange. They noted that Wladimir J. van der Laan's argument that most people are easily tricked into adding someone as a friend does not apply to friend-to-friend connections in Freenet, where adding friends is optional unlike in Retroshare where friends need to be connected.Friend-to-friend connections provide additional value for communication with real-life friends, according to the email exchange. The Freenet system builds on top of these connections, allowing for confidential messaging as an additional value. Those interested in learning more about Freenet can read "Let us talk over Freenet, so I can speak freely again" and "Freenet: The forgotten cryptopunk paradise."
