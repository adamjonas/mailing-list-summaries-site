Summary:

On March 11, 2015, Thomas Kerin proposed a new standard for creating standard multisignature P2SH addresses given m and a set of public keys. He created a PR on bitcoin/bips and used BIP0090 as a placeholder but requested a BIP number for this purpose. This method is currently used by CryptoCorp for their external signer service.
