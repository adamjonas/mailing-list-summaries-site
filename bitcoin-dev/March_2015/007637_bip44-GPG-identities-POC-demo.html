Summary:

A demonstration of using a bip44 Wallet to generate deterministic GPG identities has been implemented in javascript and is available on http://memwallet.info/bip44ext/test.html. This allows users to manage a GPG identity for encryption and signing with zero bytes of permanent storage, such as on Tails, for example. The paper can be found on https://github.com/taelfrinn/bip44extention/blob/master/README.md. A minor correction has been added specifying that the smallest S value should be used to prevent different ecdsa implementations from creating non-canonical/identical outputs. Comments are welcome.
