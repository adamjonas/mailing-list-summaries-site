Summary:

The security of a PaymentRequest is being discussed in the given context. The PaymentRequest, which is sent from a wallet to a hotel server, can be intercepted if the attacker can break TLS. The hotel server has a copy of the PaymentRequest as it is issued by them and used to validate receipts. Therefore, it is essential to ensure that the TLS protocol is being used to prevent any possible interception.
