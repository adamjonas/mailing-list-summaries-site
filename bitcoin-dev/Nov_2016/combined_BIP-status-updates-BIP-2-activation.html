Summary:

Luke has opened a pull request to implement BIP 2, which includes the activation and implementation of BIP 123. He plans to merge this on December 14th but invites people to raise any hard objections before then. Luke also welcomes further reviews of the implementation in the meantime, but requests that people avoid requesting changes to the BIPs themselves unless it is a blocker or trivial.During the implementation of BIP 2, Luke found several BIPs that were missing proper licenses in the repository. Therefore, he requested the authors of these BIPs to open a pull request and add the necessary Copyright section and License header(s). If there are other contributors to the document, he will try to reach out to them to obtain permission. In case anyone accepted contributions from someone not documented in git as an Author, they should mention this explicitly in the PR.Luke provided a list of all the BIPs that require a license, including BIP 001, 010, 011, 012, 013, 014, 015, 016, 021, 030, 031, 032, 033, 034, 035, 039, 043, 044, 045, 047, 061, 062, 064, 066, 067, 068, 070, 071, 072, 073, 075, 101, 102, 103, 106, 120, 121, and 123.
