Summary:

A covenant implementation using OP_CAT and OP_CHECKSIGFROMSTACKVERIFY op codes is possible in the Elements Alpha sidechain. The construction process has been detailed in a blog post with an example of scripts for the Moeser-Eyal-Sirer vault. Russell O'Connor, the author of the post, invites people to suggest other useful covenants for implementation. The two script extensions are already available in the Elements Alpha sidechain, making it possible to construct covenants today. Any questions regarding the implementation can be directed towards Russell O'Connor.
