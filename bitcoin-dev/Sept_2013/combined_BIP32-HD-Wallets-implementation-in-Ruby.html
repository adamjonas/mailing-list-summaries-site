Summary:

Micah Winkelspecht has developed an open source Ruby gem library called MoneyTree that implements Hierarchical Deterministic Bitcoin wallets (BIP32). The library has passed all of the test vectors and has 100% code coverage. Micah has written a detailed README which explains HD Wallets in depth, and he plans to turn it into a blog post. He is new to crypto and wants someone knowledgeable about BIP32 to review it before releasing it to the public. He invites anyone with knowledge of the BIP32 spec to check out the repo on GitHub and give feedback. Micah also welcomes anyone with conceptual accuracy knowledge to read over his README.
