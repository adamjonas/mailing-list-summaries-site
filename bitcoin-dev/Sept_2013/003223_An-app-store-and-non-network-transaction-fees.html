To use Bitcoin for managing pledges, people can either use a dedicated app or a wallet with the right features. There are pros and cons to each option. However, bitcoinj makes it easy to bring up a new GUI wallet app if written in Java. Thus, having a dedicated app for pledge management is quite straightforward. The next step is programming contracts, which can be done through the link provided for Working With Contracts on the bitcoinj wiki.
