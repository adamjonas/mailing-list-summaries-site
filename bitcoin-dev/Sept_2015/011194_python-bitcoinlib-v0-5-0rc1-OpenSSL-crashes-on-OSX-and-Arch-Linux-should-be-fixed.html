Summary:

Peter Todd, a notable figure in the Bitcoin community, has announced the official release of v0.5.0 of python-bitcoinlib on Github. The release comes after no issues were reported with the release candidate (v0.5.0rc1). The python-bitcoinlib is a Python interface to Bitcoin and other cryptocurrency-related tools using the JSON-RPC API. The library aims to simplify Bitcoin programming and is suitable for both beginners and advanced users. The release includes various features such as support for P2SH multi-sig transactions, BIP32 hierarchical deterministic key generation, as well as other improvements and bug fixes. The announcement was made on Sep 06, 2015, via email on the Bitcoin-dev mailing list.
