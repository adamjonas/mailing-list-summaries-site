Summary:

A proposed implementation of Merkleized Abstract Syntax Trees (MAST) using OP_CAT and OP_EVAL (BIP12) has been introduced. The concept was inspired by Pieter's Tree Signatures, where a scriptSig and scriptPubKey are used to execute the 10th sub-script out of 11 sub-scripts in the MAST. The suggested approach is to enable OP_CAT inside OP_EVAL for a pure softfork. This method appears to be the simplest way to implement MAST since the reference implementation for BIP12 already exists.
