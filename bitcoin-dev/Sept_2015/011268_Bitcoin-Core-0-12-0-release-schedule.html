Summary:

The next major release of Bitcoin Core, 0.12.0, is planned for the end of 2015, according to a post on the Bitcoin development mailing list by Wladimir J. van der Laan. He proposed a detailed schedule for the development of the new version: on November 1st, open Transifex translations for 0.12, and finalize and close translation for 0.10; on December 1st, feature freeze and translation string freeze; and on January 6th, split off `0.12` branch from `master`, start RC cycle, tag and release `0.12.0rc1`, and start merging for 0.13 on master branch. The aim is to release 0.12.0 final on February 1st, 2016. Van der Laan expects that in December he will probably not get much done code-wise due to Scaling Bitcoin Hongkong, 32C3, end of year festivities, etc.
