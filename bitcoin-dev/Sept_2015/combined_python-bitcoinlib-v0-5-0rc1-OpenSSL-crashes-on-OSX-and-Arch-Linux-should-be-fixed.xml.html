Summary:

Peter Todd, a prominent figure in the Bitcoin community, has announced the official release of v0.5.0 of python-bitcoinlib on Github. The python-bitcoinlib is a Python interface to Bitcoin and other cryptocurrency-related tools using the JSON-RPC API. The library aims to simplify Bitcoin programming and is suitable for both beginners and advanced users. It includes various features such as support for P2SH multi-sig transactions, BIP32 hierarchical deterministic key generation, as well as other improvements and bug fixes. The announcement was made on Sep 06, 2015, via email on the Bitcoin-dev mailing list.The new release candidate of python-bitcoinlib (v0.5.0rc1) has been released with a significant fix for OpenSSL related crashes on OSX and Arch Linux. The update also includes notable bug fixes such as getinfo() now working with disablewallet=1. The release notes list other changes including new RPC calls: getbestblockhash, getblockcount, getmininginfo; signing and verification of Bitcoin Core compatible messages with pubkey recovery; Tox tests; Sphinx docs. The release notes request confirmation that the new release candidate fixes any previously experienced OpenSSL related crashes on OSX or Arch Linux.Overall, the new release of python-bitcoinlib provides improved functionality and better stability for programmers working with Bitcoin and other cryptocurrencies.
