Summary:

The new release candidate of python-bitcoinlib (v0.5.0rc1) has been released with a major fix for OpenSSL related crashes on OSX and Arch Linux. The release notes list other changes including new RPC calls: getbestblockhash, getblockcount, getmininginfo; signing and verification of Bitcoin Core compatible messages with pubkey recovery; Tox tests; Sphinx docs. The update also includes notable bug fixes such as getinfo() now working with disablewallet=1. The release notes request confirmation that the new release candidate fixes any previously experienced OpenSSL related crashes on OSX or Arch Linux.
