Summary:

Gregory Maxwell, a contributor to the bitcoin-dev mailing list, suggested changing the process for requesting a number assignment for Bitcoin Improvement Proposals (BIPs). The current process, outlined in BIP01, is outdated due to changes in how BIPs are stored and presented. Maxwell suggested opening a pull request with the BIP text, using the author's name and an index as the number, as the new mechanism for requesting number assignments. He requested responses only to this specific question, and not to start a new thread for a different subject. Peter Todd acknowledged the suggestion with an "ACK" response.
