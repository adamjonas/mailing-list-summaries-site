Summary:

A modification proposal has been made for BIP-325, which concerns the genesis block in Bitcoin. The proposal recommends that the genesis block should be static and rely on the message start to avoid collisions between signets when multiple networks exist simultaneously. The proposed modification has been submitted through a pull request on the Github repository of Bitcoin Improvement Proposals. If implemented, this change could help prevent potential issues with signet collisions in the future.
