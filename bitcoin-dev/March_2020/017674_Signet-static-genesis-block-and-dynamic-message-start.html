Summary:

A proposal has been made to modify BIP-325 regarding the genesis block. The modification suggests that the genesis block should be made static and rely on the message start to avoid any collisions between signets when multiple networks exist simultaneously. This modification has been proposed via a pull request on the Github repository of Bitcoin Improvement Proposals.
