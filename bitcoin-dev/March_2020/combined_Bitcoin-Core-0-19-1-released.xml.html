Summary:

Bitcoin Core has released version 0.19.1 with bug fixes, performance improvements, and updated translations. This version supports and is extensively tested on operating systems using the Linux kernel, macOS 10.10+, and Windows 7 and newer. It is not recommended to use Bitcoin Core on unsupported systems. The pre-compiled distribution of this release provides binaries for the RISC-V platform in addition to previously supported CPU platforms. To upgrade, users need to shut down the older application, wait until it has completely shut down and then run the installer (on Windows) or just copy over `/Applications/Bitcoin-Qt` (on Mac) or `bitcoind`/`bitcoin-qt` (on Linux). The change log includes several improvements including a fix for origfee return for bumpfee with feerate arg, fix issue with conflicted mempool tx in listsinceblock, reset reused transactions cache, handle unspendable psbts, etc. There are also updates made for compatibility and platform support.
