Summary:

Wladimir J. van der Laan has released a new version of Bitcoin Core with an update to rc2. The previous release candidate, rc1, was "DOA," according to meeting minutes from Thursday. Binaries for the updated version are available at https://bitcoin.org/bin/bitcoin-core-0.12.1/test.rc2/, and the source code can be found on Github at https://github.com/bitcoin/bitcoin/tree/v0.12.1rc2. This release candidate includes the BIP9, BIP68, and BIP112 softfork, as well as various bugfixes and updated translations. Preliminary release notes for the release are available at https://github.com/bitcoin/bitcoin/blob/0.12/doc/release-notes.md. Release candidates are test versions for releases, and if no critical problems are found, this release candidate will become the official 0.12.1 release. Bugs should be reported using the issue tracker at Github: https://github.com/bitcoin/bitcoin/issues.
