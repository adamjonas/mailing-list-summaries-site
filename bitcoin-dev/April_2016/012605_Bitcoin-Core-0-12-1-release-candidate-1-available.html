Summary:

Bitcoin Core version 0.12.1rc1 binaries are available at https://bitcoin.org/bin/bitcoin-core-0.12.1/test.rc2/. The source code can be found on github under the signed tag at https://github.com/bitcoin/bitcoin/tree/v0.12.1rc1. It is a release candidate for a new minor version release that includes the BIP9, BIP68 and BIP112 softfork, bugfixes and updated translations. Preliminary release notes can be found at https://github.com/bitcoin/bitcoin/blob/0.12/doc/release-notes.md. Release candidates are test versions for releases. When no critical problems are found, this release candidate will be tagged as 0.12.1. Bugs should be reported using the issue tracker at github: https://github.com/bitcoin/bitcoin/issues.
