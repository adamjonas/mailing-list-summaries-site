Summary:

In certain contexts, using a file approach to download and open a Bitcoin URI may be the best option. However, this requires an additional click using a download manager, which some users may find inconvenient. A Bitcoin URI with r= eliminates the need for this extra click, but may not always be applicable. For example, when sending payment requests through messaging apps or email attachments, a file approach is likely the better choice.
