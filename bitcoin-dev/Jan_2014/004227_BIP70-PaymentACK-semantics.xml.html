Summary:

Mike Hearn, a developer, raised an important question on Bitcoin development mailing list in January 2014. He asked whether the wallet should broadcast the transaction to the bitcoin network when it receives an ACK or always assume that the merchant server will do that. He also pointed out that at that time, there was no way to differentiate between a failed/rejected submission and a successful one beyond the freeform memo field. He suggested having an error code field as well for future versions of Bitcoin.
