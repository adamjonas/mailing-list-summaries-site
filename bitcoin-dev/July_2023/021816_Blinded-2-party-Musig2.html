Summary:

In a discussion thread, Jonas Nick points out that none of the attacks mentioned so far can be prevented by proving knowledge of the signing key (PoP). The attacks discussed include an attack on the nonces and an attack on the challenge c. However, the focus should not be on 'signing', but rather on the 'secret' values, i.e., the r values (ephemeral keys). By providing proof of knowledge of the r values used to generate each R, it is possible to prevent the Wagner attack. This approach adds an extra layer of security and mitigates the potential vulnerabilities mentioned earlier.
