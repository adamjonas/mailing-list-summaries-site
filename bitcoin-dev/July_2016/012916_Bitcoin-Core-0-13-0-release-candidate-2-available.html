Summary:

Bitcoin Core version 0.13.0rc2 binaries are available on bitcoin.org. This is a release candidate for a new major version release, which includes new features, bug fixes and other improvements. The source code can be found on Github under the signed tag. Preliminary release notes are also available on Github. Release candidates are test versions for releases and when no critical problems are found, this release candidate will be tagged as 0.13.0.Notable changes since rc1 include fixes to OSX non-deterministic dmg, setting memory explicitly, adding dbcache migration path, moving SetMinVersion for FEATURE_HD to SetHDMasterKey, creating a new HD seed after encrypting the wallet, preventing fingerprinting and disk-DoS with compact blocks, exposing a flag for BIP112, scaling legacy sigop count in CreateNewBlock, and making witness v0 outputs non-standard. Users are encouraged to report bugs using the issue tracker on Github.
