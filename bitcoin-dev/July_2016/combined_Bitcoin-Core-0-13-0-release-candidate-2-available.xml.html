Summary:

Bitcoin Core version 0.13.0rc2 binaries are now available on bitcoin.org. This release candidate for a new major version release includes bug fixes, new features and other improvements. The source code for this release can be found on Github under the signed tag, with preliminary release notes also available on the same platform. Release candidates are used for testing purposes, and if no critical problems are identified, will be tagged as 0.13.0. Notable changes since rc1 include various fixes such as preventing fingerprinting and disk-DoS with compact blocks, moving SetMinVersion for FEATURE_HD to SetHDMasterKey, scaling legacy sigop count in CreateNewBlock, making witness v0 outputs non-standard, adding dbcache migration path, setting memory explicitly and creating a new HD seed after encrypting the wallet.Users are urged to report any bugs using the issue tracker on Github.
