BIP114 MAST updated

    Published on: 2016-07-31T02:03:11+00:00

  • Click here to read original discussion on the bitcoin-dev mailing list

Summary:

A new version of BIP114 MAST has been published with some added features. The updated version allows different parties in a contract to keep their scripts private until redemption, without exposing them to each other. Additionally, the new version includes a field for indicating script language version so that new opcodes can be added without affecting the version byte or the witness program. The updated BIP and code can be found on GitHub at https://github.com/bitcoin/bips/blob/master/bip-0114.mediawiki and https://github.com/jl2012/bitcoin/tree/bip114v2. However, the old version is still available at https://github.com/bitcoin/bips/blob/7478ee3260c0d3c0cef39233931b307691764edc/bip-0114.mediawiki and https://github.com/jl2012/bitcoin/tree/segwit_mast.
