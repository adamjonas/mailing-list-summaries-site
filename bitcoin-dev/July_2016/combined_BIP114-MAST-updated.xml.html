Summary:

A new version of BIP114 MAST has been released with additional features. The updated version enables different parties in a contract to keep their scripts private until redemption, without disclosing them to each other. Furthermore, the updated version includes a field for indicating the script language version which permits the addition of new opcodes without affecting the version byte or the witness program. The updated BIP and code can be found on GitHub at https://github.com/bitcoin/bips/blob/master/bip-0114.mediawiki and https://github.com/jl2012/bitcoin/tree/bip114v2. However, the old version of BIP114 MAST is still available at https://github.com/bitcoin/bips/blob/7478ee3260c0d3c0cef39233931b307691764edc/bip-0114.mediawiki and https://github.com/jl2012/bitcoin/tree/segwit_mast.
