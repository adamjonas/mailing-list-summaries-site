Summary:

The Linux builds of version 0.8.0rc1 are easily gitian-reproducible, whereas the Windows builds vary with every compile due to the absence of the -frandom-seed flag in the leveldb build. The OSX build needs a pull request to compile. The plan is to announce 0.8.0rc1 with the available binaries and perform more testing before the final release. A patch will be created to fix the non-determinism issue in Windows before the final release or rc2. Gavin Andresen plans to sign the windows setup.exe and send out an rc1 announcement the next day.
