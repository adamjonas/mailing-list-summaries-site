Summary:

Pieter Wuille, along with Peter Todd and Greg Maxwell, has proposed a plan for coordinating future soft-forking consensus changes. The proposal supports multiple parallel changes and those that get permanently rejected without obstructing the rollout of others. Wuille invites comments on the proposal but suggests using the mailing list as the gist does not support notifying participants of new comments. He plans to address feedback and work on an implementation next week.
