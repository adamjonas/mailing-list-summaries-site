In May 2015, Matt Corallo expressed his concerns to Gavin Andresen regarding the block size issue. Despite Andresen's claims that the block size would increase soon, Corallo labeled it a "huge issue". Andresen acknowledged the need for a change in the max block size but stated that there was no consensus yet on when or how it should happen. He pledged to push for a change within the year.
