Summary:

The proof-of-work (PoW) consensus protocol allows new nodes to determine the strongest chain by knowing only the genesis block and network rules. However, this does not ensure strong security guarantees since if an attacker can block connections to legitimate nodes, a new node will accept the attacker's chain. This issue is fundamental but often avoided in discussions. Bitcoin already incorporates "weak subjectivity" through checkpoints in its source code, making it difficult to take PoW purists seriously.
