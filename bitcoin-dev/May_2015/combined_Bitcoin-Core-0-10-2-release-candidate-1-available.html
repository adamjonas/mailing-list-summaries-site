Summary:

Bitcoin Core has released a new version, the release candidate 1 for the 0.10.2 version which is now available for download on the official website. Users are encouraged to test this new version before the final release. This update includes several bug fixes and improvements such as better memory usage, faster block validation, and improved error handling. The update also includes support for Tor hidden services, allowing users to run a full node on the Tor network. Additionally, this version features a new API (Application Programming Interface) that simplifies the process of building applications on top of the Bitcoin network, making it easier to develop new applications and services.Users who wish to upgrade to the new version should back up their wallet and follow the installation instructions provided on the Bitcoin Core website. It is important to note that this is a release candidate, which means it is not yet considered stable and may contain bugs. However, the release of Bitcoin Core 0.10.2 RC1 marks an important milestone in the ongoing development of the Bitcoin network. The new version offers several important improvements and features, and users are encouraged to test it out and provide feedback to the development team.
