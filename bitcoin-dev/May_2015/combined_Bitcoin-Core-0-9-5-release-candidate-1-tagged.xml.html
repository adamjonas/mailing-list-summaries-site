Summary:

The Bitcoin team has released the first candidate for version 0.9.5, tagged as `v0.9.5rc1`. This release is aimed at making BIP66 available on the 0.9 branch, which has been requested by several users, especially miners. The preliminary release notes are available on Github. However, unlike the releases of 0.10.x versions, it is uncertain whether there will be enough gitian signatures for a binary release, so any help with building is welcome.It should be noted that the release process for 0.10 or master will not work as many steps have changed since then, according to the release-process.md file.
