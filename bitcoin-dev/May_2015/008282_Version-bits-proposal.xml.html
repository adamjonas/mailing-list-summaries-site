Pieter, along with Peter Todd and Greg Maxwell, has proposed a method for coordinating future soft-forking consensus changes. The proposal, which can be found at https://gist.github.com/sipa/bf69659f43e763540550, allows for multiple parallel changes and rejected changes that will not obstruct the rollout of others. Pieter invites feedback on the proposal and recommends using the mailing list as the gist does not support notifying participants of new comments.
