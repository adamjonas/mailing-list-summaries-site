Summary:

The new release of BitCoinJ 0.5, the library for Android Wallet, SatoshiDice, Bitcoin Status and server side part of BCCAPI is now available. The new release focuses on bug fixes, completion of transition to the protobuf wallet format and standardization of build. Furthermore, it includes a native API preview, which allows the user to access bitcoinj from C++/Objective-C++. The Google+ page for announcements and developer tips/ideas has also been created. The new features include Address.getParameters(), Address.getParametersFromAddress(), updated bouncy castle version, faster transactions receiving, update to testnet2 rules, auto-migration of wallets to protobuf format, standard Maven directory layout and many bug fixes. Thanks have been given to the contributors of this release. The new release can be downloaded from www.bitcoinj.org.
