Summary:

On May 12, 2012, Jeff Garzik announced that bitcoin/bitcoin.git now supports IPv6 and several IPv6 nodes were already live on the network. The users with IPv6 were encouraged to pull the latest version and test it out. Pieter Wuille's DNS seeder running at seed.bitcoin.sipa.be has started crawling the IPv6 network, which returns corresponding AAAA records to help IPv6 nodes find each other. Previously, an IPv6-only node could not start up and sync with the network unless a dual-net node was manually created and added. However, things are now working fine out of the box, and Peter Todd had five peers when he last checked. The website sixxs.net is recommended as a resource for those without IPv6. Jeff Garzik made the announcement representing exMULTI Inc. through a message on a mailing list.
