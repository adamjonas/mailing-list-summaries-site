In a conversation between Wladimir and Miron/devrandom, the topic of discussion is building a basic virtual machine using Gitian for gitian builds. Devrandom asks if the process can be scripted, to which Wladimir replies that it can probably be done using vmbuilder or vagrant, as suggested by Nick Simpson. However, the purpose of providing exact steps is to allow anyone to follow them without needing scripting languages. The aim is to avoid ending up with a gitian-builder-that-builds-a-gitian-builder. While some parts may be scripted, Wladimir hopes that over time, Gitian itself will become easier to use and set up, requiring fewer steps.
