Summary:

The updated spec for key-stretching includes a minimum of 8192 rounds of salted PBKDF2-HMAC-SHA512 to protect passphrases, even if the bulk of the work is outsourced to another device. This is designed to allow weaker devices to export key-stretching to more powerful devices such as mobile phones and laptops. The spec has been updated to remove the Scrypt dependency and add new key derivation functions and plausible deniability. Jean-Paul Kogelman announced the updates and provided a reference implementation.
