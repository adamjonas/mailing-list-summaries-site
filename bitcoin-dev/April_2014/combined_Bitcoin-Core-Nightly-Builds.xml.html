Summary:

Bitcoin Core developers are planning to release v0.9.2 with mostly bug-fix, cleanup, and translation updates in a few weeks. Warren Togami Jr. announced this on April 16, 2014, urging power users to test the master branch using unofficial nightly builds to aid in finding bugs and polishing things up faster. Translators can also use the latest code to see what strings need to be translated as they approach the next stable release. The details can be found at https://bitcointalk.org/index.php?topic=571414.0.Cory Fields has made efforts to enable deterministic builds for MacOS X in Bitcoin Core, resulting in the availability of Windows, Linux, and MacOS X test builds for download.
