Summary:

Bitcoin-qt is issuing an unnecessary alert which is causing fear among normal users. The message, "URGENT: Upgrade required" appears to be a false positive and is motivating people to use harmful workarounds. This issue is impacting the Bitcoin community and the stable channel seems to be affected the most. Launchpad.net provides further information on this matter. Developer Oliver Egginger suggests turning off the message if possible to prevent panic and confusion. Patrick explains that the alert system filters based on the client's version number, highlighting that the ubuntu bitcoin ppa differs from the binaries available from bitcoin.org as it is dynamically linked to openssl. Jameson adds to the conversation by informing everyone that Matt just pushed out new builds to the Ubuntu PPA, stating that this issue should resolve itself shortly. The email thread ends with two links to learn graph databases and the Bitcoin-development mailing list.
