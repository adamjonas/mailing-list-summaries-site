Summary:

There is a proposal to remove the "wallet structure" part of the BIP32 document and instead point at BIP43. BIP43 defines a "purpose" and leaves everything else to the inventor of that purpose, allowing for a natural evolution of standards over time. The suggestion is to move away from a top-down approach and allow standards to develop more organically. A pull request will be submitted if there is sufficient agreement on this proposal. The link to BIP43 is provided for further reference.
