Summary:

Pieter has discovered two interesting mathematical facts about Bech32 codewords and Shamir's secret sharing. The first fact is that affine combinations of two Bech32 codewords is again a valid Bech32 codeword. The second fact is that Lagrange polynomials form a partition of unity. This means that if all shares in Shamir's secret sharing have valid Bech32 checksums, the resulting secret will also have a valid Bech32 checksum.Conversely, if both secret and random shares have valid Bech32 checksums, then all derived shares will also have valid Bech32 checksums. This discovery can be used to create a simple secret sharing scheme for dividing up a BIP-32 master seed. An example scheme for k-of-n Shamir's secret sharing is illustrated, with the aim of creating a hand-computable version of the same idea.However, this process may become obsolete with the implementation of a threshold musig signature scheme.
