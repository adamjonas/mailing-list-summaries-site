Summary:

Exchanges and miners, who are using the network 100% of the time, face the problem of getting booted off the network. This is not a concern for normal wallet users as they are not taking payments every few minutes like the exchanges. However, if miners aren't up to date, it could cause a sudden drop in hashpower that would affect the network for all users, including normal wallet users, by delaying confirmations.
