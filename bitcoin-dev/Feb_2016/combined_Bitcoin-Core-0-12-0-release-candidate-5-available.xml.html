Summary:

Bitcoin Core version 0.12.0rc5 binaries and source code are now available for download from https://bitcoin.org/bin/bitcoin-core-0.12.0/test/ and https://github.com/bitcoin/bitcoin/tree/v0.12.0rc5 respectively. The new release candidate includes new features, bug fixes, and other improvements. Detailed preliminary release notes can be found at https://github.com/bitcoin/bitcoin/blob/0.12/doc/release-notes.md. If no critical issues are found, the release candidate will be tagged as version 0.12.0. A diff since rc3 (rc4 was DOA) is also listed, including various updates such as fixing spelling errors in net.h, ensuring headers count is correct, correctly reporting high-S violations, updating the wallet best block marker before pruning, and changing solve() to use rehash in rpc-tests.Users are encouraged to report bugs using the issue tracker at GitHub: https://github.com/bitcoin/bitcoin/issues.
