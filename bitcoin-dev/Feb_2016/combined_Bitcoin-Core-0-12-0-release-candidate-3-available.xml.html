Summary:

Bitcoin Core version 0.12.0rc3 binaries and source code are now available for download from the bitcoin.org website and GitHub, respectively. This release candidate features bug fixes, new features, and other improvements. Preliminary release notes can be found on GitHub at https://github.com/bitcoin/bitcoin/blob/0.12/doc/release-notes.md. Release candidates serve as test versions, and if no critical problems are found, this release candidate will be tagged as 0.12.0. The difference since rc2 includes changes such as renaming permitrbf to mempoolreplacement, providing minimal string-list forward compatibility, updating hardcoded seeds in January 2016, not absolutely protecting local peers, adding whitelistforcerelay to control forced relaying, security/export checks to gitian, and increasing the SUBVERSION_COLUMN_WIDTH of Peertable in qt. A new certificate was also used to sign the Windows installer, which should solve compatibility issues with Windows 7. The executables were made possible thanks to the gitian builders who kept up so quickly after tagging.Any bugs encountered should be reported using the issue tracker at https://github.com/bitcoin/bitcoin/issues.
