Summary:

Bitcoin Core version 0.12.0rc5 binaries are available for download from https://bitcoin.org/bin/bitcoin-core-0.12.0/test/ and the source code can be found on GitHub under the signed tag https://github.com/bitcoin/bitcoin/tree/v0.12.0rc5. The release candidate brings new features, bug fixes, and other improvements. Detailed preliminary release notes are available at https://github.com/bitcoin/bitcoin/blob/0.12/doc/release-notes.md.The release candidate will be tagged as version 0.12.0 if no critical issues are found. A diff since rc3 (rc4 was DOA) is listed including various updates such as fixing spelling errors in net.h, ensuring headers count is correct, correctly reporting high-S violations, updating the wallet best block marker before pruning, and changing solve() to use rehash in rpc-tests.Users are encouraged to report bugs using the issue tracker at GitHub: https://github.com/bitcoin/bitcoin/issues.
