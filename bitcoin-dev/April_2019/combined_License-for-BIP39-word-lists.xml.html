Summary:

Elia inquired about using the BIP39 word lists available on Github BIP repository for their own project and raised concerns about whether or not usage for other projects is permitted due to the lack of a license agreement associated with the lists provided. Pavol Rusnak, the author of the wordlist, replied that it can be used without any restrictions and suggested the use of the SLIP39 standard for splitting shares which uses a different wordlist with better properties. The SLIP39 standard can be found on SatoshiLabs' GitHub page along with the wordlist. It is not clear who else will adopt the wordlist or whether it is live in Trezor. Peersm.com is noted as a browser version to move coins by oneself, while bitcoin-transactions and bitcoin-wallets made simple are also available on GitHub. Torrent-live.org is provided as a good resource for dynamic blocklists for anti-spies and private torrents.
