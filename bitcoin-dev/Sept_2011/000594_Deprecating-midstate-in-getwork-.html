Summary:

The internal reference miner is set to be simplified by removing the dependence on cryptopp. This change is necessary because cryptopp is only used in one place instead of OpenSSL. However, cryptopp is also responsible for calculating getwork "midstate", which is a redundant field that could easily be calculated from the blockheader. Therefore, it is proposed that this field should be removed as well.The author of the proposal is seeking thoughts and suggestions on where such changes should be announced to ensure that all miners can be upgraded.
