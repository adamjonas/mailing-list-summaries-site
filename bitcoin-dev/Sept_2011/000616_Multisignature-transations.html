Summary:

The latest update adds support for three new types of transactions, including (a and b), (a or b), and (a and b) or c. However, it is unclear whether this means that dispute mediation (2-of-3) will no longer be supported. Some users had expected the addition of CHECKMULTISIG for a small number of keys, but it is not clear if this functionality has been added with this update.
