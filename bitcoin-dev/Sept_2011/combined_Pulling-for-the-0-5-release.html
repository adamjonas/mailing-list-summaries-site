Summary:

On Monday morning, Gavin Andresen plans to pull changes for the Bitcoin 0.5 release into tip-of-git. This transition from wxwidgets to qt for the GUI is expected to cause some instability and chaos. Andresen aims to have a release candidate ready in a week or two after testing and hopes to make the 0.5 release available a month from now. However, this timeline may be affected if any major show-stopper bugs, network stability issues, denial-of-service attacks or security vulnerabilities are discovered.
