Summary:

Pieter Wuille suggested standardizing out-of-band communication inside a protocol via email or XMPP and signing the message with the "from" key. He also argued that deterministic keypairs eliminate the need for bidirectional communication and that a standard for a "public key root" "address" could be useful. These methods retain pseudonymity and allow for stateless transactions with out-of-band data.
