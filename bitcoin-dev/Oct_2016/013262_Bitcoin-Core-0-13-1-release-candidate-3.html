A new version of Bitcoin, 0.13.1rc3, has been released on GitHub with a small fix for an out-of-place assertion in rc2. Due to the simplicity of the change, it was decided that there will be no official binaries made available for rc3 and 0.13.1 should not be delayed. Users are encouraged to test the new release before the final tag on Thursday and report any bugs using the issue tracker on GitHub.
