Summary:

On December 9, 2013, Warren Togami Jr. reported a minor error in version 0.8.6rc1 of Bitcoin. The merge for the leveldb subtree was done incorrectly, resulting in an extra db/ directory in the base of Bitcoin instead of src/leveldb. Gavin Andresen confirmed the issue and noted that the file db/autocompact_test.cc should be in src/leveldb/db/, but considered it to be a non-showstopper bug. Despite having hundreds of downloads with no other reports of issues, they decided to tag and ship v0.8.6 on the same commit as v0.8.6rc1.
