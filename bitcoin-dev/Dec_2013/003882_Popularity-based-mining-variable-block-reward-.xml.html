Summary:

In an email conversation between Jameson Lopp, a software engineer at Bronto Software, and Jan Kučera in 2013, Kučera proposed a concept where there would be no reliance on external data as the network itself would decide on reward height. Each network node would determine the popularity based on various factors such as coin valuation/exchange rate, number of transactions, and many others, and come up with its block reward value.
