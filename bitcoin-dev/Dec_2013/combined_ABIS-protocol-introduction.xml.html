Summary:

The ABIS protocol is a blockchain-based identity system that allows users to securely store and share their personal information. It includes features like multi-factor authentication and flexible access controls. The protocol has been developed over several years and is now being presented as a basic idea for review and possible consideration. Further additions are expected in the near future. Interested parties can find more information on the ABIS protocol through the GitHub link provided. The creators of ABIS are seeking feedback and contributions from those interested in the development of the protocol.
