Summary:

Wladimir has made changes to Bitcoin Improvement Proposals (BIPs) and highlighted them in the given context. These changes include moving from wiki to Github as the platform for BIPs, allowing Markdown format and fixing weirdnesses such as mandatory subscription to a specific email list for BIP editors. Additionally, a new section has been created to list BIP editors, which currently only includes gmaxwell. Wladimir prefers discussions on mailing lists instead of forums. It is important to note that these changes are part of a cleanup and change process for Github move and do not affect jgarzik's proposed process. The pull request for these changes can be found on Github.
