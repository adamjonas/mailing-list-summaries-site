The Lightning Network is a Bitcoin scalability solution that facilitates secure and fast transactions. The unit for creating invoices on Lightning is millisatoshi, as stated in a referenced document. However, questions have been raised regarding the necessity of invoicing for 1000 millisatoshi for a 1 sat transaction. C-lightning is an implementation of the Lightning Network protocol.
