Summary:

The proposer suggests backdating the Pay to Script Hash (P2SH) and Version 0 Witness Program script rules back to the genesis block to simplify consensus rules. The change would allow protocol implementers to avoid writing and testing code paths that are no longer relevant. While this can be enforced with only one historical exception, it would make the consensus rules simpler. The full BIP draft can be found on Github, while the pull request to Bitcoin Core which implements this change is open and available for review.
