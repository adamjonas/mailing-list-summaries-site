Summary:

During a protocol-only discussion, Felix Wolfsteller expressed disappointment over off-list feedback regarding his proposal to make changes to the copyright-header in source files. He provided a link to follow the progress of his proposal, which involves using HTTPS instead of HTTP where possible, and suggested creating an issue on Github, 'sed -i' relevant links, and creating a pull request on Github before returning to the mailing list. This change would affect hundreds of files and may require a copyright-year-extension to 2018.The current majority of source files contain a "default" copyright header that points out where to find the MIT license. However, the link provided is using http instead of https, resulting in a redirect to https://opensource.org/licenses/mit-license.php. The proposer emphasized the importance of updating the links and shared an example header from a file in the Bitcoin codebase. Although the proposer requested quick feedback, he acknowledged uncertainty about the culture surrounding the issue.
