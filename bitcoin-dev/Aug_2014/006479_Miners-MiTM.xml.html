Summary:

In order for a miner to securely exchange information with a pool, they only need a mutual private MAC key that authenticates messages. This can be achieved through a registration step using a simple web interface over https to the miner website. Although the worst DoS could prevent new miners from joining the pool, it is not very common. The MAC key can be provided by the miner and the pool associates it with a Bitcoin public address. The overhead for this process is minimal.
