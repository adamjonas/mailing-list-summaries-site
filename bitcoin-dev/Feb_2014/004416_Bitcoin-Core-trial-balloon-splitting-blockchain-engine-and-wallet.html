In a conversation between Dustin D. Trammell and Jeff Garzik, Trammell expressed his concern about the disk space usage of running multiple wallets that require their own copy of the blockchain. He suggested running a local blockchain server or network instead to optimize disk space. Garzik agreed with Trammell's concern and mentioned that he manages multiple wallets using symlink replacement. Garzik is a Bitcoin core developer and open source evangelist at BitPay Inc.
