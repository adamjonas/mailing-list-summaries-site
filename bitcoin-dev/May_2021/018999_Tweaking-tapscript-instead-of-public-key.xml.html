Summary:

A P2TR address contains a 32-byte public key which can be used directly for creating Schnorr signatures or indirectly by revealing tapscript. This raises the question of whether it is possible to modify any taproot output after confirmation without changing the address. The base point private key with P2TR address tb1p0xlxvlhemja6c4dqv22uapctqupfhlxm9h8z3k2e72q4k9hcz7vqqzj3dz could be used by the owner to modify spending conditions without moving on-chain coins.
