Summary:

The BIP 78 PayJoin P2EP protocol allows users to fund and open channels in a single transaction, which eliminates the need for sending funds to the node on-chain and then opening channels in another transaction. This protocol pairs with the BOLT 2 Channel establishment protocol and automates PSBT channel establishment communications while avoiding surveillance heuristics. Additionally, on-chain transactions using Taproot outputs appear like regular PayJoins. An experimental alpha called "nolooking" that implements this work has been released on the Umbrel app store. With this new protocol, a brand new node can become fully connected in a single transaction that opens channels to outbound peers and immediately swap for inbound capacity by scanning a single QR code.
