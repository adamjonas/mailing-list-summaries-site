Summary:

The Ivy prototype higher-level language and development environment has been released, allowing for the creation of custom Bitcoin Script programs. The simple smart contract language aims to improve the usability of Bitcoin Script by adding features like named variables and clauses, static types, and a familiar syntax for function calls. Ivy can compile to Bitcoin Script and is intended for educational and research purposes only. The Ivy Playground for Bitcoin allows users to create and test simulated contracts in a sandboxed environment. It's important to note that Ivy should not be used to control real or testnet Bitcoins.
