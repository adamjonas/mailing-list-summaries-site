Summary:

The discussion on the Bitcoin-dev mailing list revolved around the question of how to know what consensus has formed around a UTXO set. Kalle Rosenbaum questioned the need for witnesses after a certain point, as once updates to the UTXO set are buried deep enough in the blockchain, they are no longer needed to authenticate the update. He suggested that sign-to-contract protocols should find ways to work in a witnessless environment and users could set up their own archival nodes. Eric Voskuil disagreed, stating that if nodes are not going to validate the chain, there is no reason to run a full node.
