Summary:

Bitcoin Core v24.0.1 has been updated with Antoine Riard's full-rbf peering code and minor updates to ensure that a core group of nodes are reliably propagating full-rbf replacements. The update advertises a FULL_RBF service bit when mempoolfullrbf=1 is set, and connects to four additional FULL_RBF peers. There are currently around 30 reliable nodes accepting incoming connections and running the v24.0 version of this branch, with another ~50 reliable Bitcoin Knots nodes accepting incoming connections. While Knots advertises the service bit, it doesn't have the peering code. Anyone can download the update with the provided git clone command.
