Summary:

This is a link to Bitcoin Core v24.0.1, which has Antoine Riard's full-rbf peering code and minor updates. This update ensures that a core group of nodes are reliably propagating full-rbf replacements. The update advertises a FULL_RBF service bit when mempoolfullrbf=1 is set, and connects to four additional FULL_RBF peers. It is estimated that there are currently around 30 reliable nodes accepting incoming connections and running the v24.0 version of this branch. Additionally, there are another ~50 reliable Bitcoin Knots nodes accepting incoming connections; Knots advertises the service bit, but doesn't have the peering code. The update is available for anyone to download with the provided git clone command.
