Summary:

In a discussion about URI schemes, Andreas Schildbach suggests changing the req:something parameter to one of four unreserved characters that do not need URL encoding. These options are req_something (underscore), req-something (hyphen), req~something (tilde), and req.something (period). Schildbach personally prefers the hyphen option. Matt Corallo then asks if the colon sign is allowed in unescaped/unencoded URL parameter names, suggesting an unrestricted character like underscore instead. The conversation takes place on the Bitcoin-development mailing list.
