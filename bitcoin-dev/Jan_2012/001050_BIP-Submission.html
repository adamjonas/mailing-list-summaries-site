Summary:

In a discussion about the OP_ADD function in Bitcoin, it is revealed that the function is limited to 4-byte operands and changing this would require a hard blockchain split or entire-network-upgrade. In light of this, user Ben Reeves has submitted a draft BIP for an alternative to BIP 16, or "/P2SH/". The draft can be found on the Bitcoin Wiki page. The message was sent by Gavin Andresen.
