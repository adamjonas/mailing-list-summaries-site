Summary:

The Bitcoin project website has made available for download version 0.6 release candidate 3 binaries. The update includes a BIP30 security fix set to become effective from March 15 and resolves an upgrade issue experienced while transitioning from rc1, which resulted in being stuck on the incorrect block-chain fork. This new release ensures better network security and smoother functionality of the Bitcoin network. Interested users can access the update at SourceForge's Bitcoin project website.
