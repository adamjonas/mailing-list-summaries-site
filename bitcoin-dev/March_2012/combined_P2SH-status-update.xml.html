Summary:

In the world of cryptocurrency, two proposals have been put forth for discussion: BIP16 and BIP17. BIP16 has received 37% support and only 4% opposition, while BIP17 has only received 4% support with no opposition. BIP16 aims to introduce a new script opcode that will enable new transaction types, while BIP17 suggests a new address format to improve security by preventing accidental spends. A user named "slush" inquired about Deepbit's support status, to which another user replied that nobody had discussed P2SH with Tycho recently. On March 6, 2012, Luke-Jr sent an email to the Bitcoin-development mailing list sharing updates on both proposals. The email also included an advertisement for LearnDevNow, an online learning library for Microsoft developers. While both proposals have their own merits, it remains to be seen how they will be implemented and whether they will bring positive changes to the crypto world.
