Summary:

The release of version 0.6 release candidate 3 binaries is available for download at SourceForge's Bitcoin project website. One of the main reasons to upgrade to rc3 is to obtain the BIP30 security fix that will be effective from March 15. This update also includes a fix for the issue encountered while upgrading from rc1 and being stuck on the wrong block-chain fork. The availability of this release candidate ensures better security and smooth functioning of the Bitcoin network.
