He also mentions that he intends to marketize the application in the near future, so updates will likely be made in a slightly different setup.In another email, Martinx thanks Michael for libcoin and requests that he create a regular Linux user called "blockchain" and start "bitcoind" under user "blockchain." This will allow the Bitcoin blockchain to be downloaded to the /var/lib/libcoin/bitcoin/ directory and consulted through libcoin itself. Michael responds by providing some tips.Elsewhere, a user has successfully configured and compiled libcoin but encounters an error with undefined reference when attempting to make it. The post also announces the release of the first version of the crypto currency library "libcoin," based on the bitcoin/bitcoin "Satoshi" client. Libcoin's functional code has been refactored and properly encapsulated classes, resulting in a 100% compatible drop-in replacement of the bitcoin/bitcoind client. It downloads the entire block chain 3.5 times faster than the bitcoin/bitcoind client and supports builds of static and dynamic libraries on Linux, Mac OS X, and Windows. Libcoin is licensed under LGPL v. 3 and can be used in open source as well as commercial projects.Lastly, the provided text includes contact information for Michael Gronager, PhD, Director of Ceptacle in Copenhagen, including his address, email, mobile number, and website.
