Summary:

Michael has made significant updates to the libcoin/bitcoind software. He has added NATPMP along with IDG UPnP, which allows for setup of port mapping on Apple routers and other routers supporting this protocol. The software first attempts to set up mapping using UPnP and resorts to PMP if that fails. It uses a default lease time of 20 minutes, renewed every 10 minutes. Michael included relevant portions of miniupnpc and libnatpmp in the coinNAT library to make building easier. He also added proxy capability and interested parties can view implementation details in coinChain/Proxy.h. Additionally, he made an aggressive setting for quitting searching for an IGD device at 3 seconds. The portmap option is now used instead of the upnp command line option.
