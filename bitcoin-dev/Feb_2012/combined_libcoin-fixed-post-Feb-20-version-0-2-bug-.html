Summary:

On February 20th, the Bitcoin 0.2 protocol was terminated due to no checksums on version/verack. This resulted in errors in the log files and difficulty obtaining blocks for users due to forgetting to add the date check in libcoin. The issue has since been fixed. Updates for the 0.6 protocol include new internal address classes: PubKeyHash and ScriptHash to support P2SH. A lot of other code from the 0.6 branch has also been adapted. Support for BitcoinQt is being ported to use libcoin which required a couple of interface enhancements. Further details on the updates can be found at this link: https://github.com/ceptacle/libcoin/commit/98531e971152d4fff41c30b14d58b9fd548b21db.
