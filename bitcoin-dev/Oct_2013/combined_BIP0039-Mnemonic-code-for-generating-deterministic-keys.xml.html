Pavol Rusnak revisits the wordlist and replaces around 67 words that could be offensive in some context. Slush announces that they have removed many theoretically offensive words from the BIP39 wordlist and implemented an algorithm for detecting words with similar characters and resolving duplicates. They have also added password protection and seed hardening using Rijndael cipher, which allows users to generate different BIP32 wallets by using the mnemonic with different passwords. Slush asks for constructive comments on their solution as it is intended to be used in Trezor and other wallet clients.
