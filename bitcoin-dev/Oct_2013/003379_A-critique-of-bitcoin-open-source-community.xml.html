Summary:

In October 2013, Peter Todd suggested making a GitHub repository for Bitcoin Improvement Proposals (BIPs) to make it easier to archive and update them. Jeff Garzik agreed and nominated Todd to do the conversion. Todd created a GitHub repository and copied the exact text of the BIP from the bitcoin.it wiki using MediaWiki support. Tables, images, and math are all supported by GitHub. Users can edit their pull requests or create new ones easily by forking the repository. Formatting may not be perfect, but the majority of the work has been done. Todd can be reached at 'peter'[:-1]@petertodd.org.
