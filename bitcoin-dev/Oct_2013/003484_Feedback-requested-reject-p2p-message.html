Summary:

Gavin Andresen has updated the gist with categories for reject messages. The categories are protocol syntax errors (0x01-0x0f), protocol semantic errors (0x10-0x1f), and server policy rule (0x40-0x4f). In response to a query about using varint, Gavin said that they will never run out of reject codes as only eight have been defined so far. He also explained why HTTP codes cannot be used directly for this purpose.
