Summary:

The BDB files and raw blockchain are not deterministic as blocks are not added in the same order and orphans are present. However, if one prunes up to the last checkpoint, orphans before that point can be safely removed. The idea of swapping out BDB for LevelDB is also being considered, but it is uncertain how deterministic LevelDB is.
