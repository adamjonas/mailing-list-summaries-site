Summary:

Bitcoin Core version 0.12.0rc1 binaries are now available on the official website of Bitcoin. The source code can also be found on Github under the signed tag. This is a release candidate for a new major version release that brings new features, bug fixes, and other improvements. The preliminary release notes for this version are available on Github, and when no critical problems are found, this release candidate will be tagged as 0.12.0. Users are encouraged to report bugs using the issue tracker on Github. The release candidate will undergo testing before it becomes an official release. This update is expected to bring significant changes to the Bitcoin protocol, including faster transaction processing times, an updated fee estimation system, and improved privacy features.
