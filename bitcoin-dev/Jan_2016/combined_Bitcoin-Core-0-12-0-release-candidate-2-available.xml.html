Summary:

Bitcoin Core has released version 0.12.0rc2 binaries for users to test the new features, bug fixes, and improvements before the official major release. The source code can be found on GitHub under the signed tag, along with preliminary release notes for the update. This is a release candidate that will be tagged as 0.12.0 if no critical issues are found. The diff since rc1 includes various changes such as an RPC indicating replaceable transactions, an option to set transaction replacement policy, addition of missing options help, fixing inaccurate ScriptSigArgsExpected, correcting syntax error in key parser, resolving calculation issue in Transaction View, and making rpcconsole monospace font larger on Windows. Users are requested to report any bugs using the issue tracker at GitHub.
