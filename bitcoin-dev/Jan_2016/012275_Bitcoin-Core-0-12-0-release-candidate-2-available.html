Bitcoin Core 0.12.0 release candidate 2 available

    Published on: 2016-01-25T12:09:38+00:00

  • Click here to read original discussion on the bitcoin-dev mailing list

Summary:

Bitcoin Core version 0.12.0rc2 binaries are now available for users to test it, before the release of new major version featuring new features, bug fixes and improvements. The source code can be found on GitHub under the signed tag. The preliminary release notes for the release can also be found on GitHub. This is a release candidate and when no critical problems are found, this will be tagged as 0.12.0. The diff since rc1 includes RPC that indicates which transactions are replaceable, an option to set transaction replacement policy, addition of missing options help, getting rid of inaccurate ScriptSigArgsExpected, fixing syntax error in key parser, fixing calculation issue in Transaction View and making rpcconsole monospace font larger on Windows. Users are requested to report any bugs using the issue tracker at GitHub.
