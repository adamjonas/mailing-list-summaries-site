Summary:

Jeremy Rubin has shared a post on how to create fully trustless hashrate derivative contracts that can be embedded on-chain, within channels, options, or inside of DCFMPs. These contracts can be used without CTV but are improved with it. The post does not include an analysis of the profitability of these contracts nor how to price and risk them. Rubin mentions that someone else's expertise is required for this task. He also questions if a two-sided market among miners exists. Interested readers can find the full post at https://rubin.io/bitcoin/2021/12/21/advent-24/.
