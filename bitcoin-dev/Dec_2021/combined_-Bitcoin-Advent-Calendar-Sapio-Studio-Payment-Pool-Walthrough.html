Jeremy Rubin, the developer behind Sapio Studio's GUI smart contract composer for Sapio, recently showcased the project in a post. The tool allows users to visually construct complex smart contracts without needing to manually code them. This can help to streamline the process of creating smart contracts and increase accessibility for those who may not have extensive coding experience.Despite the promising functionality of the tool, Rubin notes that there is still room for improvement and development assistance would be appreciated.
