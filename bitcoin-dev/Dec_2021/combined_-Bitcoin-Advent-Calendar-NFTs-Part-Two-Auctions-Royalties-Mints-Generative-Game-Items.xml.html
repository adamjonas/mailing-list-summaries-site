Summary:

Jeremy Rubin's latest blog post explores various exciting concepts related to non-fungible tokens (NFTs). In the post, he provides code demos of Dutch auctions of NFTs with royalties. Additionally, he discusses a novel combination of attestation chains, lightning invoices, and NFTs that can create off-chain updatable and on-chain sellable in-game items.Rubin's proposed method involves using attestation chains to establish ownership of an item off-chain, while lightning invoices are used to facilitate payments for the item. The use of NFTs allows for the creation of unique in-game items that can be easily bought and sold on-chain.The post goes into detail about how this system could work and provides sample code to demonstrate its functionality. Interested readers can find the full post on Rubin's blog at https://rubin.io/bitcoin/2021/12/19/advent-22/.
