Summary:

Bitcoin Core version 0.13.0rc3 is now available for download from bitcoin.org. This release candidate introduces new features, bug fixes and improvements. Release notes for the release are available on Github. The candidate will be tagged as version 0.13.0 if no critical problems are found. Users are encouraged to report bugs using the issue tracker at Github. Notable changes since rc2 include treating high-sigop transactions as larger rather than rejecting them, fixing p2p-feefilter.py for changed tx relay behavior, and using pre-BIP141 sigops until segwit activates for mining purposes.
