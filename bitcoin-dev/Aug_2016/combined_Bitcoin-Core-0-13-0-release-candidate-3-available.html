Summary:

The Bitcoin Core version 0.13.0rc3 has been made available for download from bitcoin.org. The release candidate offers bug fixes, new features and improvements. Release notes for the release are accessible on Github, and will be tagged as version 0.13.0 if no critical issues are found. Users are encouraged to report bugs using the issue tracker at Github. Notable changes since rc2 include treating high-sigop transactions as larger rather than rejecting them, fixing p2p-feefilter.py for changed tx relay behavior, and using pre-BIP141 sigops until segwit activates for mining purposes.
