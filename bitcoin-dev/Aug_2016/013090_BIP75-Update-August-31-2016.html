The Bitcoin Improvement Proposal (BIP) 75 has been updated with changes aimed at improving flexibility and use cases. The BIP, which is designed to facilitate secure communication between Bitcoin wallets, now includes added support for ECDSA and PGP public key infrastructure types. Additionally, versioning has been added to ensure backward compatibility and make future changes easier to manage. The updates also include minor technical details aimed at improving performance and reducing ambiguity during implementation.
