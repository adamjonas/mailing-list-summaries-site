Summary:

ZmnSCPxj has proposed an improved transaction relay which uses Bose-Chaudhuri-Hocquenghem codes for set reconciliation. The proposal can be found on GitHub minisketch. In addition, Christopher DeLucia shared a paper he produced in 2017 on IBLT applications and Bitcoin via email to the Bitcoin-dev mailing list. The author admits that they have not looked at this paper or Bitcoin since 2017 and that it was not peer-reviewed; it was only for class. The paper referenced research done by Gavin Andresen and Rusty Russell. DeLucia shared the paper with the community in case it could contribute anything. An HTML attachment was scrubbed, and a non-text attachment of the paper in PDF format was provided.
