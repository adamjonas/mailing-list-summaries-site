Summary:

Wladimir has announced the tagging of 0.9.2rc2 on Friday, which includes fixes in comparison to 0.9.2rc1. The fixes include cwallet init fix, upgrading OpenSSL to 1.0.1h, and using pnode->nLastRecv as sync score. Wladimir has stated that he has not seen any new problems reported. However, more gitian builds are still required before the final release of 0.9.2 at the end of this week.
