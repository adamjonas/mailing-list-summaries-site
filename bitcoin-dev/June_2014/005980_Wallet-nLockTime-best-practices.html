Jeff Garzik, a Bitcoin core developer and open source evangelist, has suggested considering the implementation of pull request #2340 from the Bitcoin GitHub repository. The pull request is titled "Discourage fee sniping with nLockTime" and aims to discourage users from exploiting low fees by locking transactions for a certain period using nLockTime. Garzik welcomes comments from other wallet implementors regarding this proposal. He is also associated with BitPay, Inc., a payment service provider that supports Bitcoin transactions.
