Summary:

The Bitcoin community has been discussing the separation of wallet software from the core code to create cleaner and more modular software. This would involve turning part of Bitcoin Core into libraries or a library, allowing the wallet to work as an SPV client by default without the need for a core. Full nodes and wallets have completely different usage scenarios, with full nodes being online 24/7 and wallets being online as little as possible. Bitcoind already supports SPV mode, which allows bitcoinj clients to work. The wallet would maintain its own database with private keys, addresses, balances, etc., and consume some data contained in bitcoind's database using UTXO and TXO queries.There are ongoing debates about separating the wallet code from the full node, with proponents like Jorge Timón supporting the idea for cleaner and more modular software. However, others like Wladimir express reservations about the potential burden of maintaining a competitive wallet in addition to the core consensus code. Additionally, the qt-wallet currently does not support SPV operation, so complex work should be done after the wallet is separated.In terms of SPV wallets, there is discussion about whether or not a close binding to a trusted Bitcoin core is necessary. While some believe that fewer but more trusted Electrum servers is a reasonable compromise, others argue that SPV wallets with Bloom filtering can operate without special servers. However, there are still certain limitations that require the use of trusted servers for certain applications such as exchange and payment processors.Overall, the goal is to make a p2p node as useful as possible within its resource constraints, with optional advertising of new indexes being the way to go. Tamás Blummer proposes three different uses for blockchain technology, including building consensus on the blockchain, single user wallets where SPV alone is effective, and services such as exchange and payment processors where core and indexing server talking SPV to core is the best choice. Despite ongoing debates and differing opinions, the Bitcoin community continues to work towards creating efficient and secure software for users.
