Summary:

The Bitcoin Core 0.11.0rc2 executables have been uploaded to the https://bitcoin.org/bin/bitcoin-core-0.11.0/test/ website by Wladimir. The source code can be found in the source tarball or in git under the tag 'v0.11.0rc2'. Preliminary release notes are available at https://github.com/bitcoin/bitcoin/blob/0.11/doc/release-notes.md.There have been some changes since rc1 including adding the option `-alerts` to opt out of the alert system, fixing removing of orphan transactions, supporting noncontiguous block files, using best header chain timestamps to detect partitioning, detecting and rejecting LibreSSL in configure, using the new bitcoin-detached-sigs git repo for OSX signatures in gitian, fixing scheduler build with some boost versions, fixing getbalance * 0, removing translation for -help-debug options, and removing berkeley-db4 workaround in MacOSX build docs.Thanks were given to everyone that participated in development, translation, or in the gitian build process.
