Summary:

Monetas has developed a Bitmessage address derivation method from an HD seed based on BIP-43. The proposed method is aimed at reserving a purpose code as per the BIP-43 recommendation. The details of the proposal can be found on Monetas' Github page in the bip-bm01.mediawiki file. This development could have significant implications for the Bitmessage community and may improve overall security and functionality.
