Summary:

Wladimir J. van der Laan, the Bitcoin Core binary release signing key owner, has announced that starting from version 0.11.0rc3, SHA256SUMS.asc will be signed with a new key. The new public key is 4096R/36C2E964 and has a primary key fingerprint of 01EA 5486 DE18 A882 D4C2 6845 90C8 019E 36C2 E964. This change only affects SHA256SUMS.asc; Wladimir J. van der Laan will continue using his old key for gitian and commit signing. The announcement was made via a PGP-signed message.
