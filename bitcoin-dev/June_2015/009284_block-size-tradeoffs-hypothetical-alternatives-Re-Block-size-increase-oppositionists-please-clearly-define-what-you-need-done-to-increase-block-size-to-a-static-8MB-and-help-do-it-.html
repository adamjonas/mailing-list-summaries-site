Summary:

The security model of Bitcoin is based on decentralization which prevents double spending without the use of a trusted third party. However, Adam Back's statement about decentralization being key to Bitcoin's security model is vague and lacks definition and evidence. Therefore, it is important to have a more precise and accurate problem description. This will enable one to focus on helpful questions such as the incentives that exist to increase or reduce any tendencies for nodes to collude with miners or behave as trusted third parties. MAX_BLOCK_SIZE is also related to these questions.
