Summary:

The Bitcoin Core binary release signing key owner Wladimir J. van der Laan has announced that a new key will be used to sign SHA256SUMS.asc starting from version 0.11.0rc3. The new public key is 4096R/36C2E964 and has a primary key fingerprint of 01EA 5486 DE18 A882 D4C2 6845 90C8 019E 36C2 E964. It is important to note that this change only affects SHA256SUMS.asc, as Wladimir J. van der Laan will continue using his old key for gitian and commit signing. The announcement was made via a PGP-signed message.
