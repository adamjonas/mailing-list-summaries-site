Summary:

The proposal suggests the removal of coins at specific heights, either 630000 or 840000, to be developed in the following 4-8 years. This will help maintain a consistent halving interval. The current block number (B) and how many blocks behind the coin burning block is (P) will be considered in this process. Every time a new block is mined, coins that have been inactive for a while will be burnt and added to block B's miner fees, thus keeping mining rewards up and discouraging hoarding of coins.
