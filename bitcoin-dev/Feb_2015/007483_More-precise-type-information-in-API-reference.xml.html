Summary:

In an email exchange, Dario Teixeira requested the primary author of Bitcoin.org JSON-RPC reference to provide type information in the API calls for better understanding. The author suggested submitting a pull request against the docs adding the type information needed for just one RPC call, which can later be generalized across all RPC docs. The author also provided easy editing options through GitHub and local checkout for the individual RPCs, and encouraged Dario to reach out in case of any questions or concerns.
