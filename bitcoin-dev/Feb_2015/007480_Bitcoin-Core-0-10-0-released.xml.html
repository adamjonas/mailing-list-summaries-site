Summary:

Bitcoin has released two new versions of its software, Bitcoin Core 0.10.0 and 0.13.0, both of which include various changes, enhancements and bug fixes. Bitcoin Core version 0.10 includes faster synchronization for the downloading of blocks and headers, improved signing security, watch-only wallet support, and relaxed standard script rules for P2SH redemption scripts. It also introduces new command-line options, block and transaction handling, mining and relay policy, as well as BIP 66. Version 0.13.0 introduces separate flushing modes, a locking mechanism, P2P protocol and network code changes, validation changes, build system changes, wallet changes, GUI changes, and test changes. Both updates are expected to provide improved stability and functionality for users going forward.The release notes detail the changes in the Bitcoin Core v0.10.0 release, including a new DNSseed policy and improvements to the LevelDB database. The release also includes contributions from a large number of developers and translators, including Andreas Schildbach, Gavin Andresen, Peter Todd, and many others.
