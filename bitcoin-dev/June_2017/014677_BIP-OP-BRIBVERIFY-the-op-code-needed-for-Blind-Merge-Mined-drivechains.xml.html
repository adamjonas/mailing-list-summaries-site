The goal of sidechains is to allow for reorgs without affecting the mainchain. The ratchet system would enable sidechain miners to move the sidechain forward or start a split at previous block heights while the mainchain moves forward normally. The ratchet system links the h* data from bribes to sidechain blocks by adding the sidechain block hashes to the ratchet system.
