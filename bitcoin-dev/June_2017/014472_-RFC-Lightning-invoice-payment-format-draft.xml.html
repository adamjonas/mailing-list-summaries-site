Summary:

The author, Rusty Russell, has submitted a pull request on GitHub for review of a lightning payment format. The format is bech32 encoded and includes optional parts tagged with an URI scheme, similar to the already existing "bitcoin:" scheme. The proposed format also includes a MIME type for usage in NFC NDEF messages or emails. The example provided in the pull request demonstrates how to send 2500 microbitcoin using payment hash 00010203040506...0102 to a specific recipient within a specific time frame. The author has requested wider review on the pull request.
