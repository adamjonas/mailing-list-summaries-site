Summary:

On May 19, 2016, Kristov Atlas proposed a new Bitcoin Improvement Proposal (BIP) that was assigned BIP 126. This proposal is for a Heterogenous Input Script Transaction (HIT), which would allow for a transaction to contain multiple inputs with different scripts. It's important to note that transactions are never from Bitcoin addresses, as inputs are what spend from UTXOs. Additionally, it's rare for inputs to have identical scripts.
