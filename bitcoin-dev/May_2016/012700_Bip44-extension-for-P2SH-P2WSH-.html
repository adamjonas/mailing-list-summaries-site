Summary:

The email conversation is about a scheme that is independent of the number of accounts and works with BIP44 as well as BIP43 purpose 0 or any other BIP43 purpose/layout. The advantage of this scheme is that instead of overloading the account index to indicate the type of address, the chain index is used to indicate what the specific address chain is to be used for. However, there is a major problem with this scheme as it creates confusion regarding whether a wallet supports segwit or not. To avoid this, the writer suggests having another BIPXX for segwit so that a wallet can claim it is BIP44, BIP44+BIPXX or BIPXX compatible, making it clear which wallets are compatible with it.
