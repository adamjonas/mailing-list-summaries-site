The FUD surrounding a specific Bitcoin multisig scheme has been deemed unnecessary by some experts. The M of N shared public key is generated in advance, which creates the bitcoin address. Signature fragments can then be generated offline with no communication between participants. This process is detailed in a Medium post linked within the context. Andrew Poelstra responded to speculation about the security of this process, stating that there are no non-interactive Schnorr signatures.
