Summary:

Bitcoin Core has released version 0.16.3 with a fix for CVE-2018-17144, and all network participants are being urged to upgrade as soon as possible. This vulnerability could potentially cause the Bitcoin network to crash. The release notes can be found on the Bitcoin Core website. For those who build from source, the 0.14, 0.15, 0.16, 0.17, and master branches on GitHub are also fixed. It is important for all network participants to upgrade to this latest version as soon as possible in order to prevent any potential crashes and keep the network secure. The release of this new version demonstrates the ongoing efforts to maintain and improve the security and stability of the Bitcoin network.
