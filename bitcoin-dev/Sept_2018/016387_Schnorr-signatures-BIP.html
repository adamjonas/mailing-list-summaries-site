Summary:

In an email conversation, Erik Aronesty highlighted the importance of a redistributable threshold system for security purposes. He stated that if a system is not redistributable, the loss or compromise of a single key could result in lost coins, making the system unusable. He expressed concern over Bitcoin releasing a multisig that encourages loss. However, it was pointed out that there is currently no proposal for a non-redistributable multisig for Bitcoin.
