Summary:

Pieter Wuille proposed a BIP for 64-byte elliptic curve Schnorr signatures, using the same curve as currently used in ECDSA. The draft specification of the signature scheme itself does not concern consensus rules or aggregation and is a first step towards standardizing the signature scheme. Wuille plans to work on more production-ready reference implementations and tests with other people listed in the document. If accepted, this scheme may be useful in other contexts to have a common Schnorr scheme available.
