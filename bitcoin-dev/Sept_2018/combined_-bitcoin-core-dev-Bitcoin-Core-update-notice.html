Summary:

Bitcoin Core 0.16.3 was recently released with a fix for CVE-2018-17144 and Pieter Wuille urged all network participants to upgrade to this version as soon as possible. The recommendation to upgrade still stands, even though the patch can be backported to earlier versions. Some have questioned the hype surrounding the need to upgrade immediately, but no effort has been made to correct the misinformation being spread on forums and Reddit. Those who build from source can access the fixed versions of 0.14, 0.15, 0.16, 0.17, and master branches on GitHub. A recent discussion on the Bitcoin-dev mailing list clarified that running older versions can lead to double-spends and long valid forks, so upgrading to the latest version is the best advice available. Despite some thinking the language used in the announcement was stronger than necessary, it errs on the side of protecting people from harm.
