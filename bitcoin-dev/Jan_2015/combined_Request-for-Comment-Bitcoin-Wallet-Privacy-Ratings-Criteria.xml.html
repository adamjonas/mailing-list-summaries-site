Summary:

The Open Bitcoin Privacy Project has requested public feedback on its rating criteria for Bitcoin wallet privacy. This project aims to evaluate and score various Bitcoin wallets, which will be published on their website. The deadline to provide feedback is Jan 23, 2015. OBPP members including Kristov Atlas, Justus Ranvier, Chris Pacia, and Samuel Patterson authored the current version of the criteria. Feedback can be provided through the mailing list or via GitHub. The goal of this initiative is to improve the overall privacy and security of the Bitcoin ecosystem.
