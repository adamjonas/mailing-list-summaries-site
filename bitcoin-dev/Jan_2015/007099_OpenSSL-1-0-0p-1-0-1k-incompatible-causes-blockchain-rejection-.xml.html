Summary:

In a message dated January 10, 2015, Gregory Maxwell has announced the availability of patches to enable those building from source to cope with new OpenSSL versions. The patches are available on stable branches 0.8, 0.9, and rc branch 0.10. He also provided links to fetch, build, and test each version of the software. To download Bitcoin version 0.8, users can copy and paste the provided command into their terminal and follow the instructions. The same process applies to version 0.9 and version 0.10rc1+.
