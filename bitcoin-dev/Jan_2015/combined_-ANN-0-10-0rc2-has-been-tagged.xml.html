Summary:

The latest version of Bitcoin, 0.10.0rc2, has been released with changes required for interoperability with OpenSSL 1.0.1k. Among the notable changes are improvements in robustness, bug fixes, and updates to seed IPs based on bitcoin.sipa.be crawler data. The release notes detail specific changes made to catch UTXO set read errors, introduce a 10-minute block download timeout, improve the robustness of DER recoding code, fail immediately on an empty signature, and bump OpenSSL to 1.0.1k, among other things. Instructions for fetching, building, and testing the new version are included in the release notes.
