Summary:

Bitcoin Core version 0.10.1rc1 can now be tested via binaries or source code, containing bug fixes and translation updates as release candidates are test runs for releases. Anyone using older versions of Bitcoin Core should shut down their current version before upgrading. Block files and databases are not backwards-compatible with pre-0.10 versions of Bitcoin Core or other software, but it does not affect wallet forward or backward compatibility. Those who want to downgrade smoothly should make a backup of their entire data directory. There are no notable changes in this minor release, but notable changes in version 0.10 can be found in the release notes on Github. The release includes various updates including RPC, network code, validation, build system, GUI, tests and miscellaneous items. Users can report bugs through Github's issue tracker. Credits go out to everyone who contributed to this release.
