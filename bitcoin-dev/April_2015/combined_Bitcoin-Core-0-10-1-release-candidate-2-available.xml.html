Summary:

Bitcoin Core has released version 0.10.2rc1 binaries for download from bitcoin.org. This release addresses a bug that affected Windows users who had non-ASCII characters in their data directory, and includes translation updates. Users who did not experience any issues with the previous version, 0.10.1, do not need to upgrade. Release notes for version 0.10.2 can be found on Github under the signed tag. This is a preliminary release for testing purposes and if no critical problems are found, this release candidate will be tagged as 0.10.2. Bugs should be reported using the issue tracker on Github.In addition, Bitcoin Core 0.10.1rc2 executables have been uploaded to bitcoin.org for testing purposes. The source code can be found in git under the tag 'v0.10.1rc2'. The only change in comparison to rc1 is a fix by Gavin Andresen which keeps mempool consistent during block-reorgs. Wladimir thanked everyone who participated in the gitian build process.
