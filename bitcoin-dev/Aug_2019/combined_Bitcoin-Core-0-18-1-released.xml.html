Summary:

Bitcoin Core, the software that powers the Bitcoin network, has released a new version 0.18.1 with new features, bug fixes, and performance improvements. The update can be downloaded from the Bitcoin Core website or through BitTorrent. Users running older versions should shut them down and run the installer or copy over existing files. It is worth noting that the chainstate database will be converted to a new format upon first use of version 0.15.0 or later, which may take up to 30 minutes depending on the machine's speed. Compatibility information is also provided, stating that Bitcoin Core is supported and extensively tested on Linux kernel, macOS 10.10+, and Windows 7 and newer operating systems. However, it is not recommended for unsupported systems. The release notes do not provide specific details on the new features, bug fixes, and performance improvements, but it is important for users to keep their software up-to-date in order to ensure the security and stability of the Bitcoin network.
