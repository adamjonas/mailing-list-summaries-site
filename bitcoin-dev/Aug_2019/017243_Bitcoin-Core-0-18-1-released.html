Summary:

Bitcoin Core version 0.18.1 has been released, with new features, bug fixes, and performance improvements. The update is available for download from the Bitcoin Core website or through BitTorrent. Users running older versions should shut them down and run the installer or copy over their existing files. It is important to note that the chainstate database will be converted to a new format upon first use of version 0.15.0 or later, which may take up to 30 minutes depending on the machine's speed. Compatibility information is also provided, with Bitcoin Core being supported and extensively tested on Linux kernel, macOS 10.10+, and Windows 7 and newer operating systems, but not recommended for unsupported systems.
