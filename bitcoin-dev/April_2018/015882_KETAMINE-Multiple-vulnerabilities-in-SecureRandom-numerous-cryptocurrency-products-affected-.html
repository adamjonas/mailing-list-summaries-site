Summary:

The Web Cryptography API is implemented differently in Internet Explorer, using window.msCrypto instead of window.crypto. This means that without using msCrypto.getRandomValues, high-quality entropy will not be used by any library under Internet Explorer. These details were shared by Jason Davies on his website.
