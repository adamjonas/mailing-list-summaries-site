Summary:

Electrum v3.0.6 has a new option available that allows multiple change addresses, although it is off by default. In response to a proposal around enhancing privacy through change obfuscation, Eric Voskuil corrected a common misperception regarding the cost of unspent outputs. He stated that there is no necessary additional cost to the network from the count of unspent outputs and that this perception arises from an implementation detail of particular node software.
