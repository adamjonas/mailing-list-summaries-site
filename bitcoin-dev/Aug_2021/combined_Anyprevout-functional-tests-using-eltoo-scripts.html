Summary:

Richard has implemented functional test scripts for the eltoo channel update scheme to understand BIP-118/Anyprevout (APO). These tests are based on AJ Towns' rough Taproot eltoo scripts and his APO branch of core. The tests can be accessed through Github in the "test_tapscript_eltoo" function. Additionally, Richard has written a blog post that provides detailed information on how to run the tests and includes notes that may help anyone interested in APO and eltoo.Richard is open to collaborating with others interested in eltoo and APO to extend and elaborate these tests, and any pull requests would be welcome. He expresses gratitude towards AJ and Christian Decker for their time and support in helping him understand the nuances of APO and eltoo.
