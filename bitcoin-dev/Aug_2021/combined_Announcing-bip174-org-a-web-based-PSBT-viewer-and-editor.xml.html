Summary:

The conversation begins with a user suggesting the implementation of a light and dark theme switch option and color highlighting feature in a project. The response suggests that they are not skilled at UI/UX, but are willing to give it a try. The user also mentions the possibility of a similar project for experimenting with descriptors. The response offers a link to a web-based tool for building descriptors graphically based on Scratch, which can be found on BDK's website.In addition to this, Alekos Filini, a member of the Bitcoin development community, announced the launch of bip174.org, a PSBT viewer and editor that runs in the browser. The website allows users to paste a PSBT to see its content, and whenever a change is made to one of the various fields the PSBT is automatically updated. There are a few examples built in if users just want to quickly try it out. The webapp is made entirely in Rust compiled to WASM. Contributions, feedback, and bug reports are all welcome, and the source code is available on GitHub.The Airgapped Wallet Community requested the addition of output of UR-based animated QRs to the website, as an increasing number of advanced hardware and software wallets are now supporting Airgapped UR/QR PSBT signing. There are multiple libraries in multiple languages available to support these UR QRs, making it easy to output them from the site. Additionally, some major web-based transaction coordinator services are planning to add browser scanning of PSBTs on laptops with cameras. Christopher Allen provided a video example of this in action between the Foundation Devices hardware wallet and Blue Wallet.
