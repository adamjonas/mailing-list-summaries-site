    In a discussion thread on July 30, 2011, Rick Wesson suggested putting a bounty on the test framework to encourage more testing. He recommended using http://code.google.com/p/googletest/ as his choice for a C++ framework and provided a link to http://en.wikipedia.org/wiki/List_of_unit_testing_frameworks#C.2B.2B for a list of other options. It was noted that the boost testing framework had already been implemented but lacked tests.
