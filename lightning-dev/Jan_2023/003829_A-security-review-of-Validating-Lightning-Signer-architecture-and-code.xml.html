LDK developer Antoine Riard has conducted a third-party security audit of the Validating Lightning Signer (VLS) project's core mechanisms, identifying critical vulnerabilities and attack vectors. Riard said the VLS module development was advanced enough to be functional with both deployment of LDK and CLN nodes but there were missing not-implemented critical policy rules, and invoices and payments flows still needed hardening with a consistent security model.
