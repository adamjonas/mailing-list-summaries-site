Summary:

The Nayuta team is developing a Wireshark plug-in for Lightning Network (BOLT) protocol which can decode some BOLT message. This software currently works for Nayuta's implementation (ptarmigan) and Éclair. To make interoperability test less time-consuming, the team proposes that every implementation has a compile option that enables output key information file. When ptarmigan is compiled with this option, it writes out the key information file which the Wireshark plug-in uses to decode packets. When Éclair is used, the software parses log files. The proposal aims to enable communication logs of BOLT messages on the same format (.pcap) which will be useful for interoperability test.
