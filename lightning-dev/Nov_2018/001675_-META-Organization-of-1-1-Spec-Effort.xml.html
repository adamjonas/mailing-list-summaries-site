In a mailing list discussion, Rene suggested creating a wiki page to make the process transparent and link it from README.md. He also added milestones to PRs as 1.0/1.1 and aims to clear all 1.0 PRs this week for tagging in the next meeting. The discussion thread ended with a link to the Lightning-dev mailing list.
