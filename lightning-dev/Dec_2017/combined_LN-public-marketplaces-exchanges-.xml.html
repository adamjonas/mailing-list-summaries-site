Summary:

A user inquired about using Lightning Network (LN) for creating a marketplace where users can offer to take over payment obligations of someone else and earn a little income from transaction fees. Christian Decker, a developer, responded saying that short-term loans are not supported on LN as all payments are fully backed by Bitcoin. The user then proposed an idea involving middlemen paying out funds instantly and getting their money back from the original sender after confirmation. However, ZmnSCPxj suggests that the user could run an LN node themselves, eliminating the need for a middleman. The email exchange ends with the suggestion that the idea of a marketplace for middlemen may not be feasible or beneficial in the current state of LN technology.
