Summary:

The Lightning Network Daemon (lnd) has integrated Segwit into btcd, allowing it to run with bitcoind. The chain backend in lnd is now abstracted, enabling it to operate with any of the three supported chain backends. Installation instructions can be found on Github. In response to a query from Benjamin Mord about the presence of Segwit in btcd and whether lnd can run on bitcoind, Olaoluwa Osuntokun confirms that Segwit has been merged into btcd for some time now. He invites further discussion on #lnd on Freenode. Ben also expresses interest in golang, but questions whether the layer 2/layer 1 protocol binding can be abstracted away from the implementation via SPI or other means.
