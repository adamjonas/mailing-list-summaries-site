Summary:

The Lightning Network Daemon (lnd) has abstracted the chain backend, which allows it to run with any of the three supported chain backends: btcd, bitcoind, and neutrino. Segwit has been merged into btcd for some time now and it is possible to run lnd on bitcoind. The installation instructions on the lnd documentation need to be updated. A conversation about this topic can be continued on #lnd on Freenode. Benjamin Mord asked if there is a plan for the layer 2/layer 1 protocol binding to be abstracted away from the implementation on either side via SPI or such.
