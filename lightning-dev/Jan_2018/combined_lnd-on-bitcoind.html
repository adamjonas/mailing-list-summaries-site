Summary:

In response to Ben's inquiries, Olaoluwa Osuntokun confirms that SegWit has already been merged into btcd and encourages him to check out the documentation provided on Github. He further explains that lnd, the Lightning Network Daemon, has abstracted the chain backend, allowing it to run with any of the three supported chain backends: btcd, bitcoind, and neutrino. This means that lnd can now be run with bitcoind. Regarding the layer 2/layer 1 protocol binding, Ben wonders if there is a plan to abstract it away from the implementation on either side through SPI or similar methods. While this question remains open, Olaoluwa invites them to continue the conversation on #lnd on Freenode.
