Summary:

Conner Fromknecht has shared an update regarding the implementation of a linear algorithm that reduces the total number of scalar multiplications. This algorithm is intended to replace the quadratic algorithm currently used in implementations of lightning-onion. The packet construction benchmarks show an 8x speedup, from 37ms to 4.5ms, and now uses approximately 70% less memory. Conner has made a pull request (PR) to update the recommendation and reference code for lightning-rfc. The correctness of the output has been verified against the updated BOLT 4 test vector. The description of the modified source can be found in Conner's PR to lightning-onion. The PR links for lightning-onion and lightning-rfc have also been provided.
