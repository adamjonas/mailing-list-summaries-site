In response to an inquiry by Mahesh Govind on the Lightning-dev mailing list, Christian stated that there is currently no working implementation for Ethereum. Though Raiden network has attempted to port a Lightning-like network to Ethereum, it is uncertain what the current status of that project is. The differences in the underlying blockchain make it difficult to perform a direct port; however, a Lightning-like network is still feasible for Ethereum.
