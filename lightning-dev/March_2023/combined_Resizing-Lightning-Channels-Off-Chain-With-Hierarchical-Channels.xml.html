The Lightning Network is a layer-two scaling solution for Bitcoin that allows for off-chain transactions. Recently, there have been proposals to improve the efficiency of LN transactions through new techniques such as channel factories and hierarchical channels. The lightning-dev mailing list has featured discussions on these topics, including the challenges of understanding routing to a "channel" rather than a node in the Lightning Network.
