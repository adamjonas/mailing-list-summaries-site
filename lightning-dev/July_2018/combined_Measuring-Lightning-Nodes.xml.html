Summary:

To measure the Lightning network topology, one can connect to any LN node and set initial_routing_sync flag in init message, which will prompt the peer to send the whole gossip dump containing node and channel information of every public node/channel. Alternatively, one can use listnodes and listchannels commands with lightning-cli to query their c-lightning node. For those who do not want to run a node or make anything, they can look up on explorer sites such as 1ml.com. If someone wants raw graph data without having a node, they can get it from one of the lnd nodes at https://rompert.com/networkgraph. One can monitor/measure various things like monitoring a single node or success rate of payments performed on the network.
