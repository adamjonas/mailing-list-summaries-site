Summary:

The Nayuta team, led by Hiroki Gondo, has released their mainnet implementation, Ptarmigan. This is the fourth implementation of BOLT compliance and is considered to be a reckless version. Users are cautioned to only use this software with an amount they can afford to lose. ZmnSCPxj congratulated Gondo on the release and expressed hope that many bugs would be fixed in the first mainnet release. Gondo later updated that some bugs had already been fixed after the release. The code for Ptarmigan can be found on GitHub via a link provided in the announcement email, which was sent through secure email provider ProtonMail.
