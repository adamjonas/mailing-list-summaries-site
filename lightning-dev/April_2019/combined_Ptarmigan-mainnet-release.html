Hiroki Gondo, a member of the Nayuta team, has announced the release of their mainnet implementation called "Ptarmigan." This version is recklessly implemented and is the fourth implementation of BOLT compliance. Users are cautioned to use this software with an amount they can afford to lose. ZmnSCPxj congratulated Gondo on the release and expressed hope that many bugs would be fixed in the first mainnet release. Gondo later provided an update that some bugs have already been fixed after the initial release.The code for Ptarmigan can be found on GitHub, and users can access it via this link: https://github.com/nayutaco/ptarmigan. The announcement was made through email, which was sent using ProtonMail Secure Email.
