Summary:

The c-lightning 0.10.1 release has been announced as a recommended upgrade that includes new features for users and improvements for developers. This upgrade is compulsory for users of the rebalance or drain plugins as payment secrets in invoices are now mandatory. The new features for users include `experimental-dual-fund`, Taproot support for `withdraw` and `close`, and the ability to query `listchannels` by destination as well as source. Meanwhile, developers will benefit from improvements to the manual pages and plugins, along with HSM daemon separation into libhsmd and Python bindings.There were 526 commits from 15 different authors over 114 days, including 6 first-time contributors. For more information on updates and bug fixes, the changelog provides detailed insights. It is highly recommended for users of the rebalance or drain plugins to update their systems in order to comply with the compulsory payment secrets in invoices feature.
