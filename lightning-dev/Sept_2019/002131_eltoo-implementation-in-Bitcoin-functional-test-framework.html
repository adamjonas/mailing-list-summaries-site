Richard has implemented the eltoo update scheme in the Bitcoin functional test framework to better understand and refine the Bitcoin scripts and message passing protocol for eltoo. These simulations exercise a concrete implementation of the eltoo Bitcoin scripts and explore the data flow between nodes that use eltoo to update their channel state. Richard's motivation for creating these tests is to have a framework for both understanding and refining the Bitcoin scripts and message passing protocol for eltoo.
