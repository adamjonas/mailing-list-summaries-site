Summary:

The Lightning Network white paper, specifications, and source code for lnd are hosted on the same Github page, causing confusion between the open-source protocol and Lightning Labs. Arcane Research's report on the state of the Lightning Network incorrectly referred to Lightning Labs as developing the reference client and maintaining the network standards documents repository. This led to a proposal by Fabrice Drouin to remove lnd's source code from the lightningnetwork Github page and move it to lightninglabs' Github page. The proposal aims to reduce confusion among those researching the Lightning Network and properly distinguish between the open-source protocol and its implementations.
