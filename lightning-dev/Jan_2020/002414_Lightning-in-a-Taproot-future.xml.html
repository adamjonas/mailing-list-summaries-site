Summary:

The statement made by ZmnSCPxj suggests that the implementation of Schnorr and Taproot at the base layer of Bitcoin will enable the use of payment point+scalar at the Lightning layer. It is a facetious remark, meant to point out the importance of these updates for the Lightning Network. The Lightning Network is an off-chain scaling solution for Bitcoin that allows for faster and cheaper transactions. The implementation of Schnorr and Taproot will allow for greater privacy and efficiency in Bitcoin transactions. Schnorr signatures will enable multiple inputs to be signed with a single signature, reducing the size of transactions and lowering fees. Taproot will enable more complex smart contracts to be implemented on the Bitcoin network while maintaining privacy. The Lightning Network utilizes payment channels to facilitate near-instantaneous transactions between users. Payment point+scalar refers to the encryption used in these payment channels. By utilizing this encryption scheme, the Lightning Network can ensure secure and private transactions without requiring them to be broadcasted to the entire Bitcoin network. In conclusion, the implementation of Schnorr and Taproot at the base layer of Bitcoin will have significant implications for the scalability, privacy, and efficiency of the Lightning Network. This, in turn, will make Bitcoin more usable as a currency for everyday transactions.
