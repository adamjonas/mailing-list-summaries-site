Summary:

Melvin Carvalho, a developer working on the Solid project, expressed interest in integrating the Lightning Network with Solid and creating an ontology for it. René Pickhardt suggested looking into the W3C Payment Group and LightningJoule as they are working on a recommendation for browser vendors to enable a low-level payment API. Melvin plans to create an ontology that would port some of the Lightning concepts to the semantic web and is looking for any existing work in this area or anyone interested in collaborating. The email conversation also discussed the inconsistencies in vocabulary used across different implementations of Lightning Network, with René recommending using the vocabulary from BOLT 07 to create a consistent vocabulary. They also discussed technical details about Lightning nodes and the possibility of using self-describing data.
