Summary:

Rusty Russell and Laolu both discussed the issue of how to handle cases where a node needs to straddle two chains. Rusty suggested routing A -> B -> C as normal, but having C be Litecoin or some other sidechain. However, there is concern that the sender might not realize they are changing chains. Laolu suggested allocating an extra byte in the header or using the first byte of the per-hop payload to make the chain transitions explicit.
