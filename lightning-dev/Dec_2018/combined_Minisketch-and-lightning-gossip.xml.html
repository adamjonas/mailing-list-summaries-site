Summary:

On December 11th, Rusty Russell announced a library called minisketch that could be useful for summarizing data like bitcoin transaction gossip. The library is optimized for 64-bit implementation and is best suited for gossip messages. The size of the minisketch required to be sent depends on the differences with peer rather than needing consensus between peers. If truncated, minisketches can regrow into whole ones when tossed back into the ocean. Research has been conducted on estimating the sizes of differences, which is available at https://github.com/sipa/minisketch/blob/master/doc/protocoltips.md.
