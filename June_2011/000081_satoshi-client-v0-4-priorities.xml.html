    In this context, the discussion is about adding wallet encryption and import/export features to Bitcoin v0.4 for further wallet security purposes. Sipa has already rebased his showwallet tree, which is in a functional state but missing the segfault fix. Jeff Garzik believes that wallet crypto and import/export are the two "must-have" features for v0.4, and other things like autotools can wait if they hold up the release of wallet crypto. Meanwhile, Doug Huff acknowledges that sipa's showwallet tree is not entirely ready for pulling yet.
