    Summary:

    Bitcoin Knots version 0.19.0.1.knots20200104 is now available for download from the official website. This release includes new features, bug fixes, performance improvements, and updated translations. Users are encouraged to report any bugs using the issue tracker on GitHub and subscribe to receive security and update notifications. The full release notes and change log can also be found on GitHub. Notable new features include the ability to enable BIP157 (Neutrino) to serve compact block filters to peers, experimental expansion of PSBT support to include new version and proprietary fields, and the experimental "privacy mode" option in the Overview tab to hide private information.
