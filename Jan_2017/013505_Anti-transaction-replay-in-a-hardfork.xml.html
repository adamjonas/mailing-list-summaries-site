    Summary:

    In an email exchange between Tom Harding and Johnson Lau on January 24th, 2017, Lau pointed out that their proposals for a new network were different. Specifically, Lau's proposal was opt-in while Harding's was opt-out. Harding responded by stating that Lau's proposal would require unmodified yet otherwise compatible systems to change in order to support the new network, making it unlikely to be included in new network designs. Harding suggested that an opt-out approach would be more realistic and make sense for everyone. Lau clarified that their proposals were actually the same, with the only difference being the use of a bitwise not (~) before the network characteristic byte. This meant that Lau's proposal used a bit to opt-in to a network while opting-out of all others, while Harding's proposal used a bit to opt-out of a network.
